Appearing prominently in search engines (Google, Yahoo, Bing, etc.) is the goal of 10 out of 10 sites. Ideally, this happens in unpaid (natural) searches, that is, results that appear just below the "sponsored links" and do not need constant investments.

SEO (Search Engine Optimization) is the set of techniques to improve the positioning in the natural results in search engines.

ehcommerce allows the use of various SEO techniques to improve the positioning in the natural (non-sponsored) results of the main search sites.

Here are some examples:

  1. Tag <TITLE>

    The title of each page is determined automatically according to the search made or with the selected category or product.

    One of the key information used by search engines is the <title> tag. The content of this tag is usually displayed as the title of search results on search sites and at the top of the browser. In ehcommerce, the title is generated according to the page visited. For example, the title of a detail page for a product is the name of this product, followed by the name of the store.

    You can also enter a long title, which will be displayed on the other pages. See more information on the "Store data" page of the administrative site by clicking on the question mark next to the field "Tag contents <title>".

    The news detail page uses the subject of this news in the title and the sub-title in the meta description tag.


  2. Tag configuration <META DESCRIPTION>

    These tags are used by search engines to display the most relevant information on the page, just below your TITLE. Its inclusion is automatic, according to the page visited. For example, if the page details a product, the short description field of this product will be used in this tag.

    See more information in the Store description in meta description tag section of the STORE CONFIGURATION page.


  3. Configuring the <HEAD> on custom pages (hotsites)

    In custom webdesigner pages, include <head> and </head> at the beginning of the htm file and enter the tags you want in this section, including the <title> and </title>.


  4. Configuring the <META> for proving site ownership

    Google, Bing and Yahoo! offer sites with unique features for use by webmasters. To use this site, you must prove ownership of your domain. In ehcommerce this is done through the meta tag defined by the search engines and inserted in the Additional HTML Header field of the STORE CONFIGURATION page.


  5. Content hierarchy on custom pages (use of tags <h1>, <h2> etc)


  6. Generating XML for Sitemap Protocol

    For better control of pages indexed by search engines, know the Sitemap protocol, whose use is stimulated by Google and Yahoo !.

    ehcommerce makes the automatic generation of Sitemaps, making it easier for compatible search engines to include in their indexes the main pages of the store.


  7. Generating and customizing robots.txt

    ehcommerce makes the automatic generation of the robots.txt file that is used by the main indexing robots to obtain the sitemap of the store and to know access restrictions.


  8. Custom URLs and automated URLs for auto redirection

    In ehcommerce you can create customizable URL redirects. See instructions by clicking the question mark to the right of the Redirections field on the STORE CONFIGURATION page of the administrative site.

    You can also create the link with the text of the product to be fetched between the store domain and /prod.htm.

    e.g.:

    http://www.perfumes.com.br/couro+lavanda/prod.htm

  9. Friendly URLs for products and store news.

    The links generated to the product pages, news and store hotspots are optimized for search engine indexing robots (Google, Bing, Yahoo! etc) with the use of friendly URLs. See details by clicking the question mark to the right of the Friendly URLs field on the STORE CONFIGURATION page of the administrative site.

    Examples of friendly URLs:

    
www.perfumes.com.br/prod,idproduto,3391,acqua-di-gio-pour-homme
    
www.perfumes.com.br/news,idnoticia,50,nova-rodada-de-testes-do-bol

    Custom URLs for products, categories and store news.

    Product pages, categories, and store news may have customizable addresses. See details by clicking the question mark to the right of the Custom URLs field on the STORE CONFIGURATION page of the administrative site.

    Examples of Custom URLs:

    
https://www.perfumes.com.br/acqua-di-gio-pour-homme
    
https://www.perfumes.com.br/noticia/nova-rodada-de-testes-do-bol
    
https://www.perfumes.com.br/marca/ralph-lauren

  10. Inclusion of tracking code for tracking visits and goals through the Google Analytics service


  11. Customizable 404 Error page context


  12. Store keywords in meta keywords tags specific to each category and each product in the store

    Some search engines still find the content of the meta tag keyword to be relevant to their results. You can enter specific keywords for each category and each product in the store, in the respective fields in the category and product listings. Terms must be separated by commas.

Not all of these techniques require the hiring of the optional service itself, although this is recommended.
