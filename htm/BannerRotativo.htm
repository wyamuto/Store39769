Para personalizar este exemplo, envie as imagens individuais para a pasta images, todas com o mesmo tamanho em pixels, e insira o código abaixo no arquivo em que deseja aplicar o banner rotativo (ex: home.htm):

<script>
document.write("<div id='caixa' style='width:700px; height:172px; background-color:red;'></div>");

bannerRotativo();

function bannerRotativo(){
setTimeout("mostraBanner1()",0000);
setTimeout("mostraBanner2()",4000);
setTimeout("mostraBanner3()",8000);
setTimeout("bannerRotativo()",12000);
}
function mostraBanner1(){
document.getElementById('caixa').innerHTML="<a href='http://www.fastcommerce.com.br/'><img src='/lojas/00009999/images/banner1.gif'/></a>";
}
function mostraBanner2(){
document.getElementById('caixa').innerHTML="<a href='http://www.fastcommerce.com.br/'><img src='/lojas/00009999/images/banner2.gif'/></a>";
}
function mostraBanner3(){
document.getElementById('caixa').innerHTML="<a href='http://www.fastcommerce.com.br/'><img src='/lojas/00009999/images/banner3.gif'/></a>";
}
</script>

ATENÇÃO:
  1. Altere os endereços em VERMELHO para URLs de destino de seus banners.
  2. Altere o nome dos arquivos em VERDE para URLs de destino de seus banners.
  3. Altere a largura e altura do banner nos campos em ROXO.
  4. Altere de 00009999 para o ID da loja com 8 posiçoes. Por exemplo, se o ID de sua loja é 123 coloque 00000123.
Após inserir o código, clique em "Gravar", limpe o cache da loja virtual e faça testes de navegação, clicando nas imagens rotativas.
