FC 11357 :: Como publicar a loja virtual no Facebook?

3 Caso seja solicitada a autenticação por telefone, siga os passos abaixo:

3.1 Clique em Apps Registre-se como um desenvolvedor




3.2 Confirme sua senha novamente




3.3 Aceite os termos para se registrar




3.4 Informe o número do seu celular, o código recebido através de SMS e clique em Cadastre-se




3.5 Confirme conlcuindo seu registro




3.9 Clique aqui para voltar a seguir as instruções do artigo FC 11357 :: Como publicar a loja virtual no Facebook?.
