Quando o cliente está realizando um pedido
FC$.Page FC$.Step FC$.NextStep Descrição
Register Login Order Página de login
Register Registration Order Página de cadastro
Register Confirmation Order Página de confirmação
Register Completion Order Página de pedido completado

Quando o cliente está realizando um cadastro simples
FC$.Page FC$.Step FC$.NextStep Descrição
Register Login Home Página de login
Register Registration Home Página de cadastro
Register Confirmation Home Página de confirmação

Quando o cliente vai opinar sobre um produto
FC$.Page FC$.Step FC$.NextStep Descrição
Register Login Rate Página de login
Register Registration Rate Página de cadastro
Register Confirmation Rate Página de confirmação

Quando o cliente tem que se cadastrar para poder ver produtos (lojas com níveis de acesso restrito)
FC$.Page FC$.Step FC$.NextStep Descrição
Register Login Prod Página de login
Register Registration Prod Página de cadastro
Register Confirmation Prod Página de confirmação
