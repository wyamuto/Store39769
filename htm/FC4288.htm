Colors

To register for new colors, go to the STORE CONFIGURATION page on the administrative site in the Products section:



1º Step

Check the Show colors field.


2º Step

Click the arrow to the right of the Show colors field.




3º Step

Click the [New color] option.


4º Step

Enter or Color name.


5º Step

Select the color in the color palette.


6º Step

Click the Add button.


7º Step

Click the Save button at the bottom of the STORE CONFIGURATION page.



Special descriptors

To register a special descriptor, go to the STORE CONFIGURATION page on the administrative site in the Products section:



1º Step

Fill in the name of the descriptor (e.g.: Size).


2º Step

Click the arrow to the right of the descriptor name field to register the values.




3º Step

Click the [new item] option.


4º Step

Enter the value of the descriptor (e.g.: 7,5).


5º Step

Click the Add.


6º Step

Click the Save button located at the bottom of the STORE CONFIGURATION.


Obs.: The order of the sizes register will be the same order as shown in the store. For example, for sizes register first P, then M and then G, so that these sizes are displayed in this order, which is not alphabetic.

Sign all existing sizes, even if you do not already have products with some of these sizes. Thus, when registering a product of a new size, this size will already be registered and in the correct order.



HTML Customization

To link to a particular special descriptor (e.g.: products of a certain size), use the following format:

<a href="/ListaProdutos.asp?Avancada=1&Adicional1=000000">Size 7,5</a>

Exemplo utilizando o formato de URL amigável:

 <a href="/prod,avancada,1,adicional1,000000,size-75">Size 7,5</a>

In the examples above, change 000000 by the ID of the Size P.

  • adicional1: indicates which descriptor is used (e.g: Size)

  • 000000: indicates the ID of the Size 7,5 (e.g.: 988)

  • size-75: indicates the name of the descriptor to be displayed in the URL

The link above lists all products that are registered with the Size 7,5.


Simple descriptors

To register a simple descriptor, go to the STORE CONFIGURATION page in the administrative site in the Products section:



1º Step

Fill in the name of the descriptor (e.g.: Oz/ml).


2º Step

Click the Save button located at the bottom of the STORE CONFIGURATION


Obs.: The value of the simple descriptor must be entered on the product registration page (e.g.: 100ml).
