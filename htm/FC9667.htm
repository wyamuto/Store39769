2) Link para uma determinada categoria:

Para montar este link é necessário obter o IDCategoria. Para isto passe o mouse sobre a categoria desejada e verifique que na barra inferior do navegador existe um link contendo o número da IDCategoria. Se preferir, veja este ID no site administrativo, na ficha de cada categoria.

Nos exemplos abaixo, altere 000000 para o ID da categoria desejada:

http://minhaloja.lojavirtualfc.com.br/listaprodutos.asp?idloja=99999&idcategoria=000000

Exemplo utilizando o formato de URL amigável:

http://minhaloja.lojavirtualfc.com.br/prod,idloja,99999,idcategoria,000000,nome-da-categoria


3) Link para um determinado produto:

Para montar este link é necessário obter o IDProduto. Para isto passe o mouse sobre o produto desejado e verifique que na barra inferior do navegador existe um link contendo o número do IDProduto. Se preferir, veja este ID no site administrativo, na ficha de cada produto.

Nos exemplos abaixo, altere 000000 para o ID do produto desejado:

http://minhaloja.lojavirtualfc.com.br/listaprodutos.asp?idloja=99999&idproduto=000000

Exemplo utilizando o formato de URL amigável:

http://minhaloja.lojavirtualfc.com.br/prod,idloja,99999,idproduto,000000,nome-do-produto


4) Link para determinados produtos através de texto:

Nos exemplos abaixo, altere mouse+sem+fio por texto contido no nome, referência ou descrição do produto. Espaços devem ser substituídos pelo caractere +:

http://minhaloja.lojavirtualfc.com.br/listaprodutos.asp?idloja=99999&texto=mouse+sem+fio

Exemplo utilizando o formato de URL amigável:

http://minhaloja.lojavirtualfc.com.br/prod,idloja,99999,texto,mouse+sem+fio


5) Link para determinados produtos através de descritor especial:

Para montar links para um determinado descritor especial (Ex: produtos de um fabricante), utilize o seguinte formato:

http://minhaloja.lojavirtualfc.com.br/listaprodutos.asp?idloja=99999&avancada=1&adicional1=000000

Exemplo utilizando o formato de URL amigável:

http://minhaloja.lojavirtualfc.com.br/prod,idloja,99999,avancada,1,adicional1,000000,fabricante-xyz

Nos exemplos acima, altere 000000 pelo ID do fabricante. Este número pode ser obtido através do relatório Lista de descritores especiais de produtos. Foi utilizado o parâmetro Adicional1, que é o descritor especial 1 da loja de exemplo, que lista os Fabricantes de seus produtos.



6) Link para outras páginas:

Para criar link para uma página nova, utilize o seguinte formato:

http://minhaloja.lojavirtualfc.com.br/custom.asp?idloja=99999&arq=empresa.htm

Exemplo utilizando o formato de URL amigável:

http://minhaloja.lojavirtualfc.com.br/page,idloja,99999,arq,empresa.htm,sobre-a-nossa-empresa

Nos exemplos acima, o arquivo empresa.htm deve ser enviado para a pasta htm no site administrativo.

As páginas não precisam conter as tags <html></html>, <head></head> nem <body></body>, que serão ignoradas pelo navegador. Apenas o conteúdo entre as tags <body> e </body> devem constar nas páginas adicionais.


Obs1: Para identificar o link externo que remeteu o visitante, cadastre o parceiro no site administrativo. Em seguida, inclua o parâmetro &origem=parceiro. Veja o artigo: FC 3797 :: Como identificar origens de visitação e cadastrar parceiros afiliados em minha loja?


Obs2: Se a loja tiver domínio próprio contratado no Fastcommerce (servidor virtual), pode-se substituir www.rumo.com.br por www.minhaloja.com.br em qualquer link acima, exceto cadastro.asp e track.asp, que devem ser acessadas através de https://www.rumo.com.br devido ao certificado de segurança compartilhado.



Exemplos de URL com domínio próprio:

www.lingerie.com.br/categorias.asp?idloja=159 (Página de categorias da loja Lingerie.com.br)

www.perfumes.com.br/ajuda.asp?idloja=154 (Página de ajuda da loja Perfumes.com.br)

www.sogravatas.com.br/newsletter.asp?idloja=153 (Página de newsletter da loja SóGravatas)



Obs3: Para criar links nos HTMLs personalizados da loja, utilize os exemplos de links internos, veja exemplos:
