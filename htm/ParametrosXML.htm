Parâmetros de URL aceitos pela XMLProdutos.asp

Parâmetro Descrição
IDLoja=154 Informa o IDLoja. Único parâmetro obrigatório.
Texto=palavra Exibe produtos que tenham o texto indicado.
any=1 Exibe somente os produtos cujas descrições contenham qualquer uma das palavras do parâmetro texto. O padrão é exibir somente os produtos com todas as palavras do parâmetro texto.
IDProduto=99999 Exibe apenas o produto que tenha o ID interno do produto indicado.
IDCategoria=99999 Exibe apenas os produtos que tenham o ID interno da griffe.
RamoProd=99999 Exibe apenas os produtos que tenham a classe selecionada.
PrecoDe=99,99 Exibe apenas os produtos que tenham o preço maior que o indicado.
PrecoAte=99,99 Exibe apenas os produtos que tenham o preço menor que o indicado.
SelImg=0 Define qual imagem será exibida na tag <imagem> (0=principal [padrão], 1=detalhe, 2=ampliada).
ExibeDescricao=0 Define a exibição da descrição do produto:

0=Sem descrição (exceto Google Shopping)
1=Utiliza a descrição curta
2=Utiliza a descrição longa
3=Utiliza ambas as descrição (exceto Google Shopping)
4=Utiliza descrição curta sem descritores (somente Google Shopping)
5=Utiliza descrição longa sem descritores (somente Google Shopping)

Para o formato de XML do Google Shopping, as informações dos descritores serão adicionadas à descrição nas opções 1 e 2. Para que as informações dos descritores não sejam adicionadas, utilize as opções 4 ou 5.
origem=nomeorigem Define se nos links para os produtos gerados pelo XML será inserido uma origem.
est=1 Para listar apenas produtos disponíveis com estoque de 1 ou superior.
Lancamento=1 Para listar apenas produtos em lançamento.
Promocao=1 Para listar apenas produtos em promoção.
Ref=1 Para mostrar a referência no XML dos produtos.
Peso=1 Para mostrar o peso no XML dos produtos.
PrecoZero=1 Para mostrar produtos com preço zero no XML dos produtos.
DescrMult=1 Para listar os produtos com descritores múltiplos, entre vírgulas.
Ex: P,M,G
Keywords=1 Para mostrar as keywords cadastradas no XML dos produtos.
ImgSubPai=1 Para utilizar sempre a imagem do produdo principal na tag <imagem> do sub-produto. Caso contrário é exibida a imagem do próprio sub-produto, caso exista.
NomeSubPai=1 Para utilizar sempre o nome do produto principal na tag <descrição> do sub-produto. Se o campo descrição curta do sub-produto esta preenchida o default é utilizar esta descrição.
NoSub=1 Define se serão exibidos apenas os produtos principais da loja (omitindo os sub-produtos).
DifName=2,1,3,4 Utilize este campo para que sejam adicionadas informações descritivas ao nome do produto.

1=Adiciona o descritor especial 1
2=Adiciona o descritor especial 2
3=Adiciona o descritor especial 3
4=Adiciona o descritor simples 1
5=Adiciona o descritor simples 2
6=Adiciona o descritor simples 3
7=Adiciona o código do produto
8=Adiciona o nome da categoria
9=Adiciona adicionais e nome da categoria

Informe os números na ordem desejada de inclusão das informações.

Ex: para adicionar o descritor simples 2, o descritor especial 3 e o nome da categoria, nesta ordem, informe DifName=5,3,8

Os campos adicionados ao nome serão separados por vírgula.

Ex: Hugo, Eau de Toilette, masculino, 30 ml
Mult=99,99 Multiplicador para descontos de pagamentos à vista.
Ex: para dar desconto de 20%, informe Mult=0,8
Juros=99,99 Utilizado para calcular as parcelas, que serão exibidas quando o campo XML Parc estiver preenchido na ficha do produto. Se o campo XML Parc não for informado, o XML utilizará o campo Máx Parc para exibir o parcelamento do produto, caso este campo esteja preenchido.
Ex: Para juros de 1,35% ao mês, informe Juros=1,35
Mesmo se a loja não cobrar juros, informe Juros=0 para que o parcelamento apareça no XML.
Obs: Para determinar um valor padrão de juros, preencha o campo Juros padrão da página Dados da loja, que será utilizado somente se o parâmetro juros não for informado.
UA=1 Utilize este campo para que os links gerados para as páginas de produtos sejam otimizados para os robôs de indexação dos sites de busca (Google, Bing, Yahoo! etc). Veja mais detalhes sobre as URLs amigáveis.
Format=0 Indica o formato de saída do XML. Utilize Format=1 para gerar XML no formato de feed do Google Shopping.
Brand=1 Indica qual dos descritores a loja utiliza para cadastrar a marca do produto, ou se utiliza a marca como categoria. Desta forma, será inserida a informação de marca na cláusula <g:brand> do XML gerado para o Google Shopping.

1=Utiliza o descritor especial 1
2=Utiliza o descritor especial 2
3=Utiliza o descritor especial 3
4=Utiliza o descritor simples 1
5=Utiliza o descritor simples 2
6=Utiliza o descritor simples 3
7=Utiliza a categoria

Ex: <g:brand>Sony</g:brand>
Size=1 Indica qual dos descritores é usado para informar o tamanho do produto. Desta forma, será inserida a informação de tamanho na cláusula <g:size> do XML gerado para o Google Shopping.

1=Utiliza o descritor especial 1
2=Utiliza o descritor especial 2
3=Utiliza o descritor especial 3
4=Utiliza o descritor simples 1
5=Utiliza o descritor simples 2
6=Utiliza o descritor simples 3

Ex: <g:size>30ml</g:size>
CatGoogle=Categoria Indica em qual categoria do Google Shopping os produtos que constam no XML estão associados. É recomendado gerar vários feeds XML com as respectivas categorias do Google, associados ao parâmetro de categoria da loja.

Ex: <g:google_product_category>Health & Beauty > Personal Care > Cosmetics > Perfume & Cologne</g:google_product_category>


Ex: XML dos produtos da loja 153, incluindo a descrição curta e 20% de desconto para o UOL:

http://www.sogravatas.com.br/XMLProdutos.asp?IDLoja=153&ExibeDescricao=1&mult=0,8&origem=uol
