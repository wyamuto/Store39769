Appearing prominently in search engines (Google, Yahoo, Bing, etc.) is the goal of 10 out of 10 sites. Ideally, this happens in unpaid (natural) searches, which are results that appear just below the sponsored links.

SEO (Search Engine Optimization) is a set of techniques to improve the positioning in the natural results in search engines.

ehcommerce allows the use of various SEO techniques to improve the positioning in the natural (non-sponsored) results of the main search sites.

Here are some examples:

  1. Tag <TITLE>

    One of the key information used by search engines is the <title> tag. The content of this tag is usually displayed as the title of search results on search sites. In ehcommerce, the title is generated according to the page visited. For example, the title of a detail page for a product is the name of this product, followed by the name of the store.

    You can also enter a custom default title. See more information on the Store data page of the administrative site by clicking on the question mark next to the Tag contents <title> field.

    The news detail page uses the subject of this news in the title and the subtitle in the meta description tag.


  2. Tag configuration <META DESCRIPTION>

    These tags are used by search engines to display the most relevant information on the page, just below the TITLE. Its inclusion is automatic, according to the page visited. For example, if the page details a product, the short description field of this product will be used in this tag.

    See more information in the Store description in meta description tag section of the STORE CONFIGURATION page.


  3. Configuring the <HEAD> on custom pages (hotsites)

    In custom webdesigner pages, include <head> and </head> at the beginning of the htm file and enter the tags you want in this section, including the <title> and </title>.


  4. Configuring the <META> for proving site ownership

    Google, Bing and Yahoo! offer sites with unique features for use by webmasters. To use these features you must prove ownership of your domain. In ehcommerce this is done through the meta tag defined by the search engines and inserted in the Additional HTML Header field of the STORE CONFIGURATION page.


  5. Content hierarchy on custom pages (use of tags <h1>, <h2> etc)


  6. Generating XML for Sitemap Protocol

    For better control of pages indexed by search engines, use the Sitemap protocol.

    ehcommerce creates sitemaps automatically, making it easier for compatible search engines to include in their indexes the main pages of the store.


  7. Generating and customizing robots.txt

    ehcommerce creates the robots.txt file that is used by the main indexing robots to obtain the sitemap of the store and to know access restrictions. You can customize it by sending a robots.htm file to the HTM folder of the store.


  8. Custom URLs and automated URLs for auto redirection

    You can create customizable URL redirects. See the instructions by clicking the question mark to the right of the Redirections field on the STORE CONFIGURATION page of the administrative site.

    You can also create the link with the text of the products to be searched by includind this text between the store domain and /prod.htm.

    e.g.:

    www.mystore.com/apple-watch/prod.htm

  9. Friendly URLs for products and store news.

    The links generated to the product pages, news and store hotspots are optimized for search engine indexing robots (Google, Bing, Yahoo! etc) with the use of friendly URLs. See details by clicking the question mark to the right of the Friendly URLs field on the STORE CONFIGURATION page of the administrative site.

    Examples of friendly URLs:

    
www.mystore.com/prod,productid,999999,acqua-di-gio-pour-homme
    
www.mystore.com.br/news,newsid,999999,how-to-contact-us-for-product-support

    Custom URLs for products, categories and store news.

    Product pages, categories, and store news may have customizable addresses. See details by clicking the question mark to the right of the Custom URLs field on the STORE CONFIGURATION page of the administrative site.

    Examples of custom URLs:

    
www.mystore.com/acqua-di-gio-pour-homme
    
www.mystore.com/news/how-to-contact-us-for-product-support
    
www.mystore.com/brand/ralph-lauren

  10. Customize meta keywords tags to particular categories and products in the store

    Some search engines still consider the meta tag keyword relevant to their results. You can enter specific keywords for each category and each product in the store, in their respective fields in the category and product details pages. Terms must be separated by commas.
