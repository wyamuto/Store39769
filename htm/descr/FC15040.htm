Segue o procedimento para a configuração desta opção de pagamento:


1º Passo

Acesse o site https://developer.paypal.com e faça login com a sua conta do PayPal.


2º Passo

Acesse https://developer.paypal.com/developer/applications/ para ir para a seção My Apps & Credentials do Dashboard do PayPal.


3º Passo

Clique no botão Create App:


4º Passo

Defina um nome para a aplicação em App Name e clique no botão Create App:


5º Passo

Depois que o PayPal criar a aplicação, copie em API CREDENTIALS o Client ID e Secret e insira nos respectivos campos em PayPal > Credenciais PayPal no site administrativo de sua loja no Fastcommerce.


6º Passo

Na mesma tela do PayPal onde estão as informações da aplicação, em Webhook clique em Add Webhook. Em Webhook Url preencha com a URL https://SEUDOMINIO/RetornoPPP.asp e em Event types marque All events. Clique no botão Save. Este procedimento é necessário para receber as notificações das transações (retorno) do PayPal.

ATENÇÃO: Na URL acima, altere SEUDOMINIO para o endereço da loja (ex: www.minhabelaloja.com.br).


7º Passo

Marque o campo Aceita para PayPal plus e faça pedidos de teste na loja para ver se a configuração foi feita corretamente.

Obs: Para usar o PayPal Plus solicite ao PayPal a liberação deste recurso em sua conta no PayPal.
