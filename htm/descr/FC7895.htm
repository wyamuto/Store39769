Here are step-by-step to register these sub-categories:

  1. Check the "Subcategories" field on the "STORE CONFIGURATION" page of the administrative site.

  2. Register the parent category, which in this case is "Shoes", through the "PRODUCTS / Add categories" menu.

  3. Register the subcategory, which in this case is "Running", in the "PRODUCTS / Add categories" menu. Select the parent category "Shoes".

  4. To register the other subcategories, repeat "step 3".

  5. See the final result of your subcategory structure in the "PRODUCTS / Show categories" menu:


After following this procedure, the products can be registered in their respective subcategories.
