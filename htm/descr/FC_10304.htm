Para incluir estes botões (Facebook, Twitter, Google +1, etc), basta incluir na loja as tags especiais para redes sociais e marcar o campo Usar tags especiais na seção Redes sociais da página Dados da loja do site administrativo.

Exemplo 1:


<table><tr>
<td><b>Compartilhe: </b></td>
<td><ShareFacebook></td>
<td><ShareTwitter></td>
<td><ShareGooglePlus></td>
<td><SharePinterest></td>
</tr></table>

Exemplo 2:


<table><tr>
 <td><b>Compartilhe: </b></td>
 <td><Facebook type="like" send="true" layout="button_count" width="150" show_faces="false"></td>
 <td><GooglePlusOne></td>
</tr></table>

Exemplo 3:


<table><tr>
<td>
<table>
<tr><td><b>Compartilhe: </b></td></tr>
<tr><td align=center>
<table width=100%>
<tr><td><ShareFacebook></td><td><ShareTwitter></td><td><SharePinterest></td></tr>
</table>
</td></tr>
</table>
</td>
<td width=60 align=center><Facebook type="like" layout="box_count"></td>
<td width=60 align=center><GooglePlusOne size="tall"></td>
</tr></table>

Exemplo 4 - No detalhe do produto:


<div>
<ul>
<li><span><ShareFacebook></span>
<span><ShareTwitter></span>
<span><ShareGooglePlus></span>
<span><SharePinterest></span>
</li>
</ul>
</div>

Para que estes botões sejam exibidos nas páginas de detalhes dos produtos, as tags devem ser inseridas entre as tags <ProdDet> e </ProdDet>,no arquivo EstiloProduto.htm

Clique aqui para ver a lista das tags especiais para redes sociais.
