É possível exibir os dados da cesta em qualquer página da loja e em qualquer posição, como por exemplo na barra direita.

Veja abaixo exemplo de design com os dados da cesta, exibido através de um iframe contendo um XML formatado por XSL:

Seguem instruções de implantação deste recurso:

1º Passo

Marque o campo Exibe dados da cesta na página Dados da loja do site administrativo.

2º Passo

Adicione o código abaixo no local onde os dados da cesta serão exibidos (ex: BarraLateral.htm):

<span id="idMostraDadosCestaFC"></span>

3º Passo

Adicione o código abaixo ao final do arquivo Rodape.htm:

<script>
// Mostrar cesta apenas quando tiver produtos adicionados
function MostraDadosCestaFC(){
var PosCarrinho=document.getElementById("idMostraDadosCestaFC"); 
PosCarrinho.innerHTML="<div id='MostraDadosCestaIDFC' style='display:none'><table width='350' border='0'><tr><td align='center'><iframe frameborder=0 scrolling=no src=XMLCart.asp?IDLoja=<IDLoja>&style=xsl style=width:350;height:50></iframe></td></tr></table></div>";
}
MostraDadosCestaFC();

var QtdCesta="<QtdCesta>";
if (QtdCesta>0){
document.getElementById("MostraDadosCestaIDFC").style.display='';
}
</script>

4º Passo

Clique aqui com o botão direito do mouse e selecione a opção de Salvar. Envie o arquivo salvo para a pasta htm, com o nome XMLCart.xml. Este é o arquivo que irá determinar o design do XML e poderá ser personalizado pelo webdesigner da loja.

Obs: O XML do carrinho lista somente os 5 primeiros produtos existentes no carrinho, além do valor total de todos produtos adicionados e a quantidade de itens.
