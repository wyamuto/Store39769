INSTRUCTIONS:

1 Step

Register a product called Gift Card $ 50.

Obs.:The product name must match the value of the desired Gift Card.

2 Step

For each approved order, register a new coupon in the voucher value, put 1 in the Maximum use field and mark the Active field.

Obs¹.: This voucher/coupon can not be associated with a specific customer, anyone can use it.

Obs².: For better financial control, we suggest that coupon names be Gift card #35978. Where #35978 is the order number on which the Gift card was purchased.



ATTENTION: This feature may cause differences in your financial statement.

e.g.: The gift card was purchased in January, the store received this value but did not send products, if this gift card is used in March the store will deliver the product without having received $ 1.00, as this value had already been received.

More information:

In this article, the instructions are for the stores that will send the gift card via Post Office, Carrier, etc..

For stores that will deliver the gift card online (e.g.: email) we suggest you create preformatted messages for orders with Delivered status by entering the gift card code.

In the case of stores that use the subproduct feature, each gift card value must be registered as a subproduct.
