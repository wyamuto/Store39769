Para a criação de um hotsite, crie um arquivo html com o layout e envie para a pasta htm. Após a criação deste arquivo, inclua um link para esta página personalizada com o parâmetro IDCategoria da categoria que deseja apresentar nesta navegação.

Ex¹: (sem URL amigável)

<a href="custom.asp?idloja=<idlojanum>&idcategoria=165012&arq=perfumes.htm">Perfumes</a>

Ex²: (com URL amigável)

<a href=""/page,idloja,<idlojanum>,idcategoria,165012,arq,perfumes.htm">Perfumes</a>

Onde: 165012 é o ID da categoria escolhida e perfumes.htm é o arquivo html criado para este Hotsite.

Clique aqui para ver os hotsites de loja exemplo que utilizam este recurso. Em seguida, clique nos links em Lojas especiais na barra esquerda. Note que os menus da loja se alteram de acordo com a navegação.
