Na seção Facebook da página Dados da loja no site administrativo, existem dois ícones para visualizar a loja. Um deles exibirá a loja no "modo normal" e o outro exibirá a loja com o layout do Facebook.

Caso queira, é possível incluir um link para que o visitante acesse a loja com o design normal, através do parâmetro &fbh=0.

Exemplo de URL para a home da loja no design normal:

http://www.perfumes.com.br/home.asp?IDLoja=154&fbh=0

Exemplo de URL para a home da loja no design Facebook:

http://www.perfumes.com.br/home.asp?IDLoja=154&fbh=1

Para incluir este link no rodapé da loja, altere o arquivo FBHrodape.htm.

Exemplo de link:

<a target=_top href=http://www.minhaloja.com.br/home.asp?IDLoja=99999&fbh=0>Versão Clássica</a>

Altere onde esta http://www.minhaloja.com.br para o endereço da loja e 99999 para o ID da loja.
