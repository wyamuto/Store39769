Clique aqui para ver este recurso em funcionamento.

Seguem instruções para a utilização da simulação do frete na página de detalhe do produto:


1º Passo

Efetue um backup dos arquivos BarraTopo.htm e EstiloProduto.htm que sofrerão alterações nesta implantação.


2º Passo

Clique aqui com o botão direito do mouse e selecione a opção de Salvar. Envie o arquivo salvo para a pasta htm.


3º Passo

Adicione o código abaixo no começo do arquivo BarraTopo.htm:

<script language=JavaScript src=<BaseLoja>htm/SimulaFreteFC.js></script>

4º Passo

Adicione o código abaixo entre as tags <ProdDet>...</ProdDet> do arquivo EstiloProduto.htm

<div id=""ShowCEP<IDProduto>""></div>
<script>ShowCEP(<IDProduto>);</script>

5º Passo

Se o código abaixo não se encontrar no arquivo BarraTopo.htm ele deve ser adicionado.

<script type=""text/javascript"">
IDLoja='<IDLoja>'; //Obtém IDLoja para montar links
IDLojaNum='<IDLojaNum>';
</script>

ATENÇÃO: Com este recurso, é possível exibir todas as opções de frete que a loja oferece. Entretanto, para maximizar as vendas da loja, o exemplo acima exibe somente o menor valor de frete do produto. Isto melhora a usabilidade da loja, evitando que o cliente tenha que tomar decisões de frete neste momento.
