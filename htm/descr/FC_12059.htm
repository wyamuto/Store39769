Ao utilizar este recurso, quando um produto é adicionado à cesta, uma função javascript é acionada e exibe uma mensagem informando que o produto foi adicionado à cesta. Exemplo:

É possível personalizar a ação que irá ocorrer após a inclusão do produto à cesta. Para esta personalização a loja deve inserir uma função na biblioteca javascript da loja com o nome ShowCartOnPage. (Clique aqui para ver exemplo da função padrão)

Exemplo:

function ShowCartOnPage(IDLoja,iErr,sMsg,sCartText,sCheckoutText,este){
  ...
  ...
  ...
}

Onde:

  • IDLoja: É o ID da loja. Ex: 999999

  • iErr: retorna 0 se produto adicionado à cesta. Caso contrário retorna código de erro.

  • sMsg: retorna mensagem com status da inclusão do produto à cesta:

  • Um ou mais produtos não puderam ser adicionados à cesta.

  • Produto adicionado ao carrinho.

  • Produtos adicionados ao carrinho.

  • Nenhum produto foi selecionado para inclusão no carrinho.

  • sCartText: retorna texto "Ir ao carrinho". (Apenas quando não ocorre erro na inclusão [iErr=0])

  • sCheckoutText: retorna texto "Finalizar pedido". (Apenas quando não ocorre erro na inclusão [iErr=0])

  • este: retorna objeto que foi clicado para adicionar produto à cesta. Este objeto pode ser utilizado por exemplo para pegar a posição onde foi feito o click para adição do produto à cesta.

Atualizar cesta de produtos no design da loja via AJAX

É comum a loja utilizar este recurso em conjunto com o recurso que exibe os dados do carrinho no design da loja em AJAX. Desta forma, quando o cliente clicar no botão para comprar o produto, o produto será adicionado à cesta sem sair da página, irá aparecer a mensagem de que o produto foi adicionado à cesta (função ShowCartOnPage) e uma outra função (MostraDadosCesta) irá automaticamente atualizar a quantidade e valor da cesta no design da loja via AJAX.

Exemplo:

BarraTopo.htm - Design com os dados da cesta de produtos.

<script>IDLojaNum='<IDLojaNum>';</script>

<table width="290" border="0"><tr><td class="StyProdLinkCart" align="left"><a href="AddProduto.asp?IDLoja=<IDLoja>"><b>Carrinho de compras: <span class="StyProdShoppingCartPrice" id="QtdItensCesta"></span></b></a></td></tr><tr><td> </td></tr><tr><td class="StyProdLinkCart" align="left"><a href="AddProduto.asp?IDLoja=<IDLoja>"><b>Sub-total: <span id="ValorItensCesta"></span></a></b></td></tr></table>

Rodape.htm - Função que exibe os dados da cesta no design do topo via AJAX

<script>MostraDadosCesta();</script>
