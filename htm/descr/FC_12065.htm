É possível exibir um ícone informando que seu frete é gratuito:


Segue o passo-a-passo:

1º passo:

Insira a função abaixo no final do arquivo LojaLib.js:

function MostraFreteGratis(iPeso){
  iPeso=parseFloat(iPeso.replace(",","."));
  if(iPeso==0)document.write("<img src='"+FC$.PathImg+"btfretegratis.gif'>");
}

2º passo:

Crie uma imagem com o nome BtFreteGratis.gif e envie para a pasta images.

Segue exemplo da imagem utilizada neste artigo:


3º passo:

Para esta imagem aparecer na página principal da loja, o código abaixo deve ser adicionado no arquivo EstiloProdutoOut.htm. Para a imagem aparecer na lista de produtos, o arquivo é EstiloProduto.htm.

<script>MostraFreteGratis("<PesoNum>");</script>

ATENÇÃO: Para utilizar este recurso, marque a opção de Frete gratuito para peso zero na página "Envio & frete" do site administrativo. Em seguida, informe peso zero na ficha de cada produto que terá frete gratuito. Certifique-se também de que o campo Exibe peso que fica na página "Dados da loja" esteja marcado.
