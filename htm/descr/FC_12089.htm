1º Passo

No site administrativo da loja virtual, clique em Arquivos e acesse a pasta htm.


2º Passo

Localize o arquivo EstiloProduto.htm e clique no ícone de lápis para editar o código.


3º Passo

Faça uma cópia do código em um arquivo de bloco de notas.


4º Passo

No código do arquivo EstiloProduto.htm, entre as tags <ProdDet> ... </ProdDet>, localize a tag <Preco>. Se não encotrar, procure pela tag <PrecoSimples>. Em seguida, substitua a tag pelo seguinte código:

<script>
 if(<PrecoNum>>0) document.write("<PrecoSimples>");
 else  document.write("<img src='/lojas/00099999/images/consulte.gif'>");
</script>

Altere 00099999 para o ID da loja com 8 posiçoes. Por exemplo, se o ID da loja é 123, coloque 00000123.


5º Passo

Clique em Gravar.


6º Passo

Clique novamente em Arquivos, acesse a pasta images e clique para enviar o arquivo de imagem consulte.gif.


7º Passo

Após enviar a imagem, limpe o cache da loja virtual e execute testes de navegação.
