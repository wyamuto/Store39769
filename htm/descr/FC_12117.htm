Utilize os seguintes relatórios do Fastcommerce para ver as quantidades de acessos à loja.

Marque o filtro "Pessoas na loja" para excluir as visitas dos robôs dos resultados dos relatórios:

  • Acessos diários no período

  • Acessos mensais no período

Segundo o próprio Google, podem ocorrer diferenças entre os relatórios de acesso por vários motivos adicionais:

  • Os visitantes podem ter definido suas preferências de navegador de forma que o Google Analytics não consiga coletar os dados.

  • Os visitantes podem estar com javascript, cookies ou imagens desativados. Nesse caso, o Google Analytics não conseguirá informar dados sobre esses visitantes. Para que o Google Analytics registre uma visita, é necessário que o computador do visitante tenha javascript, imagens e cookies ativados para seu website.

  • Um erro de javascript de uma página personalizada da loja pode impedir a contabilização desta página pelo Google Analytics.

  • Latência do servidor: Se um visitante chegar ao site e mudar de página antes do código de acompanhamento ser executado, os dados não serão transmitidos para os servidores do Google.
