Even with this caution, your password will be exposed to third parties if you access the control panel through a computer infected by a virus or malicious program created for this purpose. To prevent this from occurring, you can add an additional level of user authentication, which is the request for a One-Time Passwords (OTP). This password is generated by a free OTP application installed on your mobile phone and can only be used once, in a short period after its generation.

To access the control panel, the user who has this feature enabled will have to enter his normal password and then will have to enter the disposable password generated by the OTP application on his cell phone. If this one-time password is captured by a third party, it will be useless because it can not be used again.

Follow the step-by-step guide to configure this important security feature. This procedure only needs to be done once:

  1. Install the password generation application (OTP) on your phone or tablet, according to your device type (iPhone, Android, Windows Phone, Java etc).

  2. Access the User password page on the control panel of the virtual store. Enter your user password and leave the remaining fields blank.

  3. In the One-Time Password (OTP) section, enter the OTP Secret field that will be informed by the mobile application.

  4. In the mobile application, enter the 4 digits entered in red in the PIN field and ask for the respective password.

  5. Enter the disposable password in the Password field of the One-Time Password (OTP)

  6. Press the SAVE button to activate the feature.

On your next access to the control panel, after entering your normal access password, you will need to enter the PIN in the mobile application, which in turn will respond with a disposable password. This additional password must be entered in the Password field of the control panel so that the user can complete their login.

ATTENTION: Always use the user with the OTP feature enabled when accessing the control panel from a shared computer, such as a lan house. A good practice is to register an additional user to be used outside of your personal computer, with OTP enabled and more restricted access.
