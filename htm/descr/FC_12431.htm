Cada página pode incluir até 4 widgets de produtos, sendo que cada widget pode exibir até 6 produtos da loja.

O parceiro poderá escolher o design, as cores e os produtos desejados, através dos parâmetros do widget.

Veja o passo-a-passo para criar um widget:


1º Passo:

Acesse a ficha do parceiro e preencha os campos Origem e Site. Em seguida, clique no botão Criar widget.


2º Passo:

No campo Tipo escolha um dos modelos pré-definidos.


3º Passo:

Escolha a opção de cores.


4º Passo:

Para apresentar determinados produtos, selecione a Categoria desejada e marque os campos Apenas promoções e Apenas lançamentos.


5º Passo:

Também é possível divulgar os produtos que contenham o termo desejado.

Ex: Informe o termo câmera digital no campo Texto

Para divulgar um produto específico preencha o campo ID do produto.


6º Passo:

Também é possível divulgar os produtos que contenham o termo desejado.

Clique no botão Exibir widget para ver com ficou o widget.


7º Passo:

Clique no botão Selecionar código para copiar o script e cole onde a divulgação será feita (sites institucionais, blogs, fan pages, etc).
