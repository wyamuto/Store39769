O exemplo de formulário desta documentação possui os seguintes campos:

  • Nº Pedido

  • Nome

  • E-mail

  • Telefone

  • Motivo


Obs: É obrigatório o preenchimento de todos os campos.


Obs: A solicitação será enviada para o mesmo e-mail que recebe as confirmações de pedido.


Seguem os passos para adicionar este formulário na loja virtual:


1º Passo

Adicione o código abaixo ao final do arquivo Rodape.htm:

<script type="text/javascript">
if (FC$.Page == "Track"){
var sFormCanc=document.getElementById('idRecoverOrderFC');
if(sFormCanc != null){
document.getElementById('idRecoverOrderFC').innerHTML+="<br /><br /><br /><a href='<htmloja>FormCancelamento.htm'>Clique aqui</a> para solicitar o cancelamento do 
pedido.";
}
}
</script>
