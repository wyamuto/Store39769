Veja o procedimento para implantação deste recurso:


1° Passo

Faça o backup dos arquivos Home.htm e Rodape.htm (Clique aqui para mais instruções sobre a geração de backups).


2° Passo

Dentro da pasta htm, crie uma pasta chamada popnews.


3° Passo

Clique aqui para baixar o arquivo PopNews.zip, que deverá ser enviado para a pasta htm > popnews.


4° Passo

Dentro da pasta images, crie uma pasta chamada popnews.


5° Passo

Clique aqui para baixar o arquivo PopNewsImg.zip, que deverá ser enviado para a pasta images > popnews.


6° Passo

Adicione o código abaixo ao início do arquivo Home.htm.

<!-- PopNews -->
<script language="JavaScript" src="/jquery-1.8.2.min.js"></script>
<script type="text/javascript" src="<BaseLoja>htm/popnews/jQueryCookie.js"></script>
<script type="text/javascript" src="<BaseLoja>htm/popnews/popnews.js"></script>
<link type="text/css" rel="stylesheet" href="<BaseLoja>htm/popnews/popnews.css">

<div id="FullBackground"></div>
<div id="popupNews">
<a href="#" class="btnClose">
<img src="<BaseLoja>images/popnews/closePopup.png" width="24" height="24" alt="" border="0">
</a>
<h1><NomeLoja></h1>
<span>Cadastre-se e aproveite nossas ofertas incríveis!</span><br>
<form class="formNews" id="formPopup" name=FormNewsletter method=GET action="Newsletter.asp" onSubmit="return ValidaNew();"/>
<input type=hidden name=IDLoja value="<idloja>">
<input type=hidden name=Y value="<Y>">
<input type=hidden name=FormatoHTML value="True">
<input type=hidden name=EnviarAssinante value="Enviar">
<input type="text" required name=NomeAssinante placeholder="Nome">
<input type="email" required name=Email placeholder="Seu e-mail"><br><br>
<input type="submit" class="bt_news btnSubPop" name=ok value="" />
</form>
</div>
<script type="text/javascript">PopNews();</script>
<!-- PopNews -->

Obs: O formulário de assinante será apresentado uma vez ao dia para cada visitante.


Para alterar este comportamento, é necessário alterar o número de dias no arquivo popnews.js, dentro da pasta htm > popnews.

Segue trecho de código que deve ser alterado:

days:1, // Define o tempo em dias para a exibição do popup

hours:0, // Define o tempo em horas para a exibição do popup

minutes:0 // Define o tempo em minutos para a exibição do popup
