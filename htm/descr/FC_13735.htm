O jQuery torna mais simples a navegação pelo HTML, a seleção de elementos DOM, a criação de animações, a manipulação eventos e o desenvolvimento de aplicações AJAX.

Para habilitar e utilizar o jQuery nos scripts da loja, insira uma das linhas abaixo no início do arquivo BarraTopo.htm, de acordo com a versão que pretende utilizar:

jQuery 2.1.1

<script type="text/javascript" src="/jquery-2.1.1.min.js"></script>

jQuery 2.0.2

<script type="text/javascript" src="/jquery-2.0.2.min.js"></script>

jQuery 1.10.1

<script type="text/javascript" src="/jquery-1.10.1.min.js"></script>

jQuery 1.10.3

<script type="text/javascript" src="/jquery-ui-1.10.3.custom.min.js"></script>

jQuery 1.10.0

<script type="text/javascript" src="/jquery-ui-1.10.0.custom.min.js"></script>

jQuery 1.9.2

<script type="text/javascript" src="/jquery-ui-1.9.2.custom.min.js"></script>

jQuery 1.9.1

<script type="text/javascript" src="/jquery-ui-1.9.1.custom.js"></script>

jQuery 1.8.2

<script type="text/javascript" src="/jquery-1.8.2.min.js"></script>

jQuery 1.7.2

<script type="text/javascript" src="/jquery-1.7.2.min.js"></script>

jQuery 1.7.1

<script type="text/javascript" src="/jquery-1.7.1.min.js"></script>

Se possível, utilize sempre a versão mais recente do framework, caso esta seja compatível com os plugins jQuery que a loja utiliza.
