O Bootstrap foi criado para ser o framework do Twitter, que o liberou em 2011 como open source para toda comunidade de desenvolvedores.

Para habilitar o Bootstrap, insira as linhas abaixo no início do arquivo BarraTopo.htm:

<script type="text/javascript" src="/js/bootstrap.min.js"></script>
<link type="text/css" rel="stylesheet" href="/css/bootstrap.min.css" media="all">

Veja mais detalhes na documentação oficial: Introdução Bootstrap
