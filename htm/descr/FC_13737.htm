O Foundation surgiu como um projeto para desenvolver código front-end mais rápido e melhor. Ele é fornecido como open source pela licença do MIT.

Para habilitar o Foundation, insira as linhas abaixo no início do arquivo BarraTopo.htm:

<script type="text/javascript" src="/js/foundation.min.js"></script>
<link type="text/css" rel="stylesheet" href="/css/foundation.min.css" media="all">

Veja mais detalhes na documentação oficial: Introdução Foundation
