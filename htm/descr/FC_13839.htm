O Fastcommerce possui um recurso onde ao pesquisar produtos que não existem na loja, ao invés de um retorno de apenas "Não foram encontrados produtos na busca" você pode criar um arquivo HTML personalizado, apontando outros produtos ou com informações para auxiliar na busca do seu cliente. Para isso veja os passos abaixo:


1º Passo

O arquivo que deve ser criado para ser exibido na loja deve ter o nome ProdNotFound com extensão HTM ficando ProdNotFound.htm. Nesse arquivo você pode adicionar nossas tags padrões e personalizar como achar melhor.


2º Passo

Após criar o arquivo é preciso enviá-lo para loja. Acessando o menu Arquivos e entrando na pasta htm, após acessar basta clicar na opção "Enviar arquivos", escolher o arquivo que criou no passo anterior e enviar para loja.


3º Passo

Faça um teste do arquivo que enviou fazendo uma pesquisa por um termo que não tenha retorno algum na sua loja e verifique se o conteúdo do arquivo ProdNotFound.htm será exibido.


Obs: Caso não tenha conhecimento para criar o arquivo abaixo você pode baixar um arquivo de exemplo. Este arquivo deve ser enviado para a pasta htm através do gerenciador de arquivos da loja.
