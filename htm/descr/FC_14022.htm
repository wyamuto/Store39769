Disponibilizamos os seguintes tipos de código:

  • Código 128A

  • Código 128B

  • Código 128C

  • Código 39

  • Código 2de5

  • EAN-13

  • EAN-8

  • UPC-A


Obs: O tipo de código de barras mais comum no varejo mundial, incluise no Brasil, é o EAN-13, que utiliza 13 dígitos. Este formato também é utilizado pelo Google Shopping. Veja exemplo:
