Segue o passo-a-passo para gerar os links manualmente:


1º Passo

Acesse o site administrativo da loja e clique em Relatórios no menu esquerdo.


2º Passo

Execute o relatório Lista de clientes e produtos de cestas abandonadas.


3º Passo

Copie o ID gerado na coluna Sessão.

Exemplo: 456987123


4º Passo

Copie o endereço abaixo e cole no seu navegador, para ver a cesta abandonada:

Domínio próprio:

http://www.sualoja.com.br/addmult.asp?idloja=99999&idr=9999999999

Sem domínio próprio:

http://sualoja.lojavirtualfc.com.br/addmult.asp?IDLoja=99999&idr=9999999999

5º Passo

Cole o ID da Sessão no parâmetro "idr" conforme o 3º passo.

Exemplo:

http://www.sualoja.com.br/addmult.asp?idloja=99999&idr=456987123

Verifique que o produto que foi abandonado está na cesta.


6º Passo

Repita o mesmo procedimento para os demais clientes ou utilize a integração via API para automatizar este processo.



Obs:

1) Nos links acima, altere o domínio e o ID loja.

2) Se a Restrição a novos clientes for maior que 2 o cliente precisa estar logado na loja.
