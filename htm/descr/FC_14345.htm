Se sua empresa fornece um sistema de remarketing, é necessário que crie uma loja gratuita de integrador no Fastcommerce. Esta loja será utilizada para realizar testes de integração e para sanar as dúvidas com a nossa equipe.

É possível fazer a integração de forma manual, inserindo scripts nos arquivos da loja (HTML e JS) ou fazer a integração via Google Tag Manager (recomendável).


Para fazer a integração via Google Tag Manager, veja os artigos abaixo:

15031 :: Como funciona a integração com o Google Tag Manager?

15032 :: Como importar as variáveis e acionadores padrões do Fastcommerce para o Google Tag Manager?



Segue o passo-a-passo para fazer esta integração de forma manual:


1º Passo

Clique aqui para criar sua loja gratuita de integrador. Esta loja ficará no ar durante o período necessário para você fazer os testes e ver como funciona o Fastcommerce.


2º Passo

Defina em quais páginas seu javascript será inserido.

Para incluir o script em todas as páginas, insira o script no arquivo Rodape.htm

Conheça as tags especiais do Fastcommerce que podem ser utilizadas neste arquivo: Tags especiais


3º Passo

Caso queira incluir seu script somente na página de detalhes do produto, insira-o no arquivo EstiloProduto.htm dentro da tag <ProdDet></ProdDet>.

Conheça as tags que podem ser utilizadas neste arquivo: Tags especiais de produtos


4º Passo

Caso queira incluir o script somente na página final de confirmação do pedido, insira-o no arquivo ConfirmacaoPedido.htm.

Conheça as tags que podem ser utilizadas neste arquivo: Tags especiais para confirmação de pedido


5º Passo

Após a inclusão dos scripts nas páginas desejadas, faça diversos testes na loja para verificar se toda a integração funciona corretamente.


Caso persistam dúvidas, entre em contato com nossa equipe do suporte, enviando mensagem para suporte@Fastcommerce.com.br.
