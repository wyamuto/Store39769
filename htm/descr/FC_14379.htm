Outros requisitos para este recurso:

  • O campo Subprodutos da página "Dados da loja" do site administrativo deve estar marcado.

  • O subproduto deve ter pelo menos um descritor cadastrado
    (ex. Tamanho: P).

  • Ao cadastrar cores com nomes compostos, coloque + ao invés de espaços entre os nomes
    (ex. Amarelo+Azul)

  • Cada combinação de tamanho e cor deve ser cadastrada como um subproduto
    (ex. Cadastre um subproduto para o tênis UltraMega - Tamanho 37 - Cor Preta, outro subproduto para o tênis UltraMega - Tamanho 38 - Cor Preta, etc)

  • Devem ser cadastradas a Imagem principal, Imagem de detalhe e Imagem ampliada em todos os produtos e subprodutos, mesmo que as imagens sejam iguais.

  • Para otimizar o zoom, sugerimos que as imagens tenham as seguintes dimensões:
    Imagem detalhada: entre 200px de largura x 200px de altura até 350px de largura x 350px de altura
    Imagem ampliada: entre 600px de largura x 600px de altura até 800px de largura x 800px de altura

var FC_QtdDescritores=2;


Caso tenha mais descritores, aumente este número. O valor máximo é 7, que é a cor mais os 6 descritores da página Dados da loja.
