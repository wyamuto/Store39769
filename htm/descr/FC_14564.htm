Investir no design responsivo coloca a loja à frente da concorrência e normalmente aumenta suas vendas.

O framework Wave CSS é utilizado para obter o design responsivo de inúmeras lojas virtuais. O projeto é continuamente desenvolvido e mantido pela equipe Fastcommerce.

Para habilitar o Wave CSS, insira a linha abaixo no início do arquivo BarraTopo.htm:

Wave CSS: versão 2.03.2015

<link rel="stylesheet" href="/wave/V2/waveMin.css">

Wave CSS versão 1.11.2014

<link rel="stylesheet" href="/wave/V1/waveMin.css">

Veja mais detalhes na documentação oficial Introdução ao Wave CSS
