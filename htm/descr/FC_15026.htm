1º Passo

Acesse o site administrativo da loja virtual e clique no link Arquivos, que fica no topo.

Na pasta HTM encontre o arquivo chamado LojaLib.js e edite esse arquivo.

Ao final deste arquivo adicione o código abaixo:

function showUser(user){ 
  sF$.fnLoginUserName(user.fullName,user.pictureURL);
}

Grave a alteração.


2º Passo

Determine onde gostaria de incluir as opções de login em sua loja. Segue um exemplo implementado no topo de uma loja fictícia. Esta implementação é feita inserindo o código abaixo:

  
<img src='<PathTo file="images/GoogleLogin.svg">' class="GoogleSigninClass" data-loginsuccess="showUser" width="24" height="24" style="background-color:#fff;margin-left:10px;">
  <img src='<PathTo file="images/FacebookLogin.svg">' class="FacebookSigninClass" data-loginsuccess="showUser" width="24" height="24" style="background-color:#fff;">

Copie o código acima e cole no arquivo BarraTopo.htm, que também fica na pasta HTM.

Escolha o local onde gostaria de inserir o código de acordo com o design da loja. Neste exemplo o código foi inserido junto aos links para acesso de Meu cadastro, Meus pedidos e Contato.



Grave a alteração.
