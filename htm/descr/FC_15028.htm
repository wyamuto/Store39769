2º Passo

URL do feed

Exemplo de URL para preencher no campo URL do feed:

http://minhaloja.lojavirtualfc.com.br/xmlprodutos.asp?IDLoja=99999&format=1&selimg=1&juros=0&idint=1

O parâmetro &format=1 indica que o XML será gerado no formato exigido pelo Facebook.

O parâmetro &idint=1 indica que serão utilizados os IDs internos do banco de dados no identificador de produtos. Ex: <g:id>5163158</g:id>

OBS1: No exemplo acima altere o ID da loja 99999 para o ID da loja no Fastcommerce. Altere minhaloja de acordo com o campo Endereço Internet preenchido na seção Navegação da página Dados da loja do site administrativo. Caso a loja tenha o serviço de domínio próprio contratado no Fastcommerce, altere o domínio minhaloja.lojavirtualfc.com.br para o domínio da loja (Ex: www.minhaloja.com.br).

Exemplo de URL: http://www.sogravatas.com.br/xmlprodutos.asp?idloja=153&format=1&selimg=1&juros=0&idint=1


OBS2: É possível montar a URL para o feed através do site administrativo de sua loja virtual. Veja mais detalhes no artigo de ajuda abaixo:

FC 3796 :: Qual a URL para exibir a lista dos produtos da loja em XML?

Para gerar o conteúdo no padrão do Facebook, selecione o formato Facebook no formulário de geração da URL do feed.


OBS3: Ao gerar a URL para o feed, utilize o parâmetro "brand" para que as marcas dos produtos possam ser enviadas ao Facebook e preencha os campos MPN (código atribuído pelo fabricante) e código de barras nos produtos da loja.


3º Passo

Aplicação no Facebook

É necessário ter uma aplicação criada no Facebook. Para criar esta aplicação, acesse https://developers.facebook.com/


4º Passo

Associação do catálogo ao recurso do Pixel

Caso a loja utilize o recurso do Pixel Facebook, associe o catalogo de produtos ao pixel clicando no botão fontes associadas no catalogo de produtos no Facebook.
