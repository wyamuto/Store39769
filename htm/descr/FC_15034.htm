1. Backup dos arquivos da loja: baixe facilmente uma cópia de todos os arquivos que compõem o design da loja, incluindo os arquivos html, js, css, imagens de design e fotos de produtos.

  2. Imagens de produtos: Uma imagem vale por mil palavras. Mas esta imagem precisa ser adequada para sua finalidade. Com a integração ao GitHub, será possível trabalhar individualmente cada imagem e atualizá-las todas de uma só vez na loja.

  3. Descrições de produtos: Caso a loja utilize as descrições de produtos em arquivos html, esta integração será excelente para a manutenção deste arquivos. Os arquivos poderão ser editados localmente e publicados rapidamente na loja.

  4. Renovação no design: o GitHub oferece diversas facilidades para gerenciar as versões de cada arquivo. Por exemplo, ao renovar a home da loja para o Natal, a versão anterior é mantida para facilitar sua publicação, após o término das vendas natalinas.

  5. Implantação de novo design completo: Com o GitHub, o deploy (publicação) de novo design ficará muito mais rápido.

A integração é ativada através do botão Integrar com GitHub, que fica no final das páginas do gerenciador de arquivos, no site administrativo da loja:
