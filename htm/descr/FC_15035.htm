Como implantar o AMP?

Siga os passos abaixo para integrar sua loja virtual ao AMP:

  1. Clique aqui para baixar um ZIP com os arquivos HTM do AMP Design. Em seguida, envie este arquivo ZIP para a pasta HTM da loja.

  2. Clique aqui para baixar um ZIP com as imagens do AMP Design. Em seguida, envie este arquivo ZIP para a pasta IMAGES da loja.

  3. Envie o logotipo da loja com o nome amplogotopo.png para a pasta IMAGES. Esta imagem deverá ter o tamanho exato de 180 x 54 pixels. Ela será apresentada no topo de todas as páginas da loja nos acessos por smartphone.

  4. A integração ao AMP é ativada através da seção AMP Design, que fica na página Dados da loja, no site administrativo:

    Selecione a opção (3) HTML + links + backlinks (preview) e grave. Nesta opção as páginas AMP não serão indexadas pelo Google. Por isto ela deve ser utilizada somente para fazer testes com o AMP Design.

  5. Para ver o design das páginas habilitadas para o AMP design, selecione cada uma das opções do campo Visualizar e clique no ícone . Veja abaixo alguns exemplos:

  6. Se o design estiver OK, selecione a opção (6) HTML + links + backlinks e grave. Em breve as páginas da loja com o design AMP serão indexadas pelo Google.



Clique aqui para mais detalhes sobre o procedimento de ativação e personalização do AMP. Esta personalização poderá ser feita pelo próprio lojista ou pelo webdesigner da sua preferência, e deverá seguir as especificações do AMP HTML.



Recomendações

Para aproveitar melhor este recurso, é recomendável que o cadastro de produtos respeite as seguintes condições:

  • O tamanho das fotos de todos os produtos devem ter as mesmas proporções (ex: quadradas, 16x9, 4x3).

  • As descrições curtas dos produtos devem estar preenchidas.


Caso a loja publique notícias, siga as diretrizes abaixo:

  • Caso a notícia tenha imagem cadastrada, é recomendado que ela tenha o tamanho exato de 696 x 390 pixels. Este é tamanho para evitar distorções na imagem e para que a mesma seja utilizada no carrossel de notícias dos resultados de busca do Google.

  • Para seguir as diretrizes de logotipo do AMP, envie a imagem logoamp.png para a pasta IMAGES da loja com o tamanho de 200 x 60 pixels.



ATENÇÃO: O checkout do pedido não é realizado no design AMP. Portanto é importante que a loja tenha a versão mobile ativada ou seja responsiva, para que as páginas de checkout sejam corretamente apresentadas em tela pequena. Entre em contato com nossa equipe de suporte caso persistam dúvidas.
