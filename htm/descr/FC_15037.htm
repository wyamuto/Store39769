Siga os passos abaixo para implementar a permissão para o Google enviar mensagem para o cliente avaliar a loja:


1º Passo

Acesse o Google Tag Manager.


2º Passo

Clique no nome do contêiner que deseja editar.


3º Passo

Clique em Tags > Nova.

  • No título, coloque Google Avaliações do Consumidor
  • Em configuração da tag, escolha o tipo da tag HTML personalizado.
  • No conteúdo da tag, coloque o código abaixo:

<script>
Date.prototype.addDays = function(days) {
  this.setDate(this.getDate() + parseInt(days));
  return this;
}

function fnDateGoogle(){
  var DiasEstimativaEntrega=7;
  var currentDate=new Date();
  currentDate.addDays(DiasEstimativaEntrega);
  var month=currentDate.getUTCMonth()+1;
  var day=currentDate.getUTCDate();
  var year=currentDate.getUTCFullYear();
  day=day+"";
  month=month+"";
  if(day.length==1)day="0"+day;
  if(month.length==1)month="0"+month;
  var newdateGoogle=year +"-"+ month +"-"+ day;
  return newdateGoogle; 
}

var DateGoogle=fnDateGoogle();

F$.fnLoadScript('https://apis.google.com/js/platform.js?onload=renderOptIn',true);

window.renderOptIn = function() {
  window.gapi.load('surveyoptin', function() {
    window.gapi.surveyoptin.render(
      {
        "merchant_id": 9999999,
        "order_id": "{{FC_orderNumber}}",
        "email": "{{FC_email}}",
        "delivery_country": "BR",
        "estimated_delivery_date": DateGoogle
      });
  });
}

window.___gcfg = {
  lang: 'pt_BR'
};
</script>


OBS¹: Altere o 9999999 do merchant_id para o código do merchant_id de sua loja. Este código pode ser visto no menu esquerdo do Google Merchant (ID do comerciante).

OBS²: Na integração deve ser enviado ao Google a data de estimativa de entrega do pedido. Na variável "DiasEstimativaEntrega" está configurado o valor 7, ou seja, a data enviada para o Google de estimativa de entrega é a data de hoje mais 7 dias. Se desejar, altere este valor de acordo com o maior prazo que o consumidor costuma receber os pedidos da loja.


4º Passo

Em Acionamento, selecione o acionador FC Evento Pedido Efetuado.


5º Passo

Clique em Salvar e depois em Publicar.


6º Passo

Faça pedidos de teste. Na página de confirmação do pedido, deve ser exibida uma solicitação do Google para perguntar se o cliente deseja avaliar posteriormente a loja.


Implementação do Selo / badge

Opcionalmente é possível inserir um Badge, que é um selo para mostrar que sua loja virtual participa do programa Google Avaliações do Consumidor e exibir sua classificação de vendedor na sua loja.

Para inserir, adicione o snippet de código abaixo a qualquer página do seu site alterando o merchant_id de 9999999 para o merchant_id de sua loja. Após adicionar o snippet de código, verifique se o selo aparece no seu site corretamente.

<script src="https://apis.google.com/js/platform.js?onload=renderBadge" async defer></script>

<script>
  window.renderBadge = function() {
    var ratingBadgeContainer = document.createElement("div");
    document.body.appendChild(ratingBadgeContainer);
    window.gapi.load('ratingbadge', function() {
      window.gapi.ratingbadge.render(ratingBadgeContainer, {"merchant_id": 9999999});
    });
  }
</script>
<script>
  window.___gcfg = {
    lang: 'pt_BR'
  };
</script>

OBS: Se não houver classificações do vendedor, a mensagem Nenhuma classificação disponível será exibida.

Normalmente, sua loja precisa ter pelo menos 150 avaliações no último ano para que a sua classificação de vendedor apareça no selo. Por isto, sugerimos que insira o selo apenas quando a loja tiver atingido este número de avaliações.
