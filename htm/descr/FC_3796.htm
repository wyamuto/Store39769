Seguem as opções para a geração desta URL:


Opção 1

Clique no item Produtos do menu esquerdo do site administrativo:


Após entrar na página Gerenciamento de categorias & produtos clique no link XML de Produtos.


No formulário para a geração de XML é possível determinar quais produtos serão listados ou listar todos os produtos da loja se nenhum campo for marcado.


Clique no botão Exibir URL e copie a URL apresentada ou utilize a opção "Acessar esta URL no navegador".


Opção 2 (avançado)

Exemplo de URL para listar todos os produtos da loja em XML:

https://minhaloja.lojavirtualfc.com.br/xmlprodutos.asp?IDLoja=99999&origem=nomeorigem

O parâmetro &origem=nomeorigem é opcional e serve para que os links no parceiro para os produtos da loja venham com a origem na URL. Caso "nomeorigem" esteja cadastrado como um parceiro no site administrativo, será possível identificar a origem da visitação através dos relatórios de parceiros. É muito útil para verificar o retorno de ações de marketing e divulgação em Shoppings virtuais e outros portais. Veja funcionalidade de Parceiros.

É possível filtrar os produtos que são exibidos no XML.


Exemplo:

Parâmetro Texto (Texto=palavra)

URL para listar apenas produtos que contenham a palavra específicada preenchendo o campo "Texto"

  • https://minhaloja.lojavirtualfc.com.br/xmlprodutos.asp?IDLoja=99999&origem=nomeorigem&Texto=livro


Parâmetro IDProduto (IDProduto=00000)

URL para listar apenas o produto com ID interno preenchendo o ID no campo "ID do produto"

  • https://minhaloja.lojavirtualfc.com.br/xmlprodutos.asp?IDLoja=99999&origem=nomeorigem&IDProduto=00000


Parâmetro Est (est=1)

URL para listar apenas produtos disponíveis com estoque de 1 ou superior denifidos na opção "Estoque mínimo"

  • https://minhaloja.lojavirtualfc.com.br/xmlprodutos.asp?IDLoja=99999&origem=nomeorigem&est=1


É possível combinar vários parâmetros: (produtos em lançamento que contenham o texto "reebok")

  • https://minhaloja.lojavirtualfc.com.br/xmlprodutos.asp?IDLoja=99999&origem=nomeorigem&Lancamento=True&Texto=reebok



Parcelamento no XML

Parâmetro Juros (juros=0)

Para exibir parcelamento no XML utilize o parâmetro Juros (ou preencha o campo "Juros (% a.m.)") e na ficha de cadastro de cada produto, informe o campo XML Parc

  • https://minhaloja.lojavirtualfc.com.br/xmlprodutos.asp?IDLoja=99999&origem=nomeorigem&juros=0


OBS 1: Nos exemplos acima altere o ID da loja 99999, para o ID de sua loja no Fastcommerce.

OBS 2: Altere minhaloja de acordo com o campo Endereço Internet em Navegação da página Dados da loja do site administrativo.

OBS 3: Caso a loja tenha o serviço de domínio próprio contratado no Fastcommerce, você deverá alterar o domínio minhaloja.lojavirtualfc.com.br para o domínio de sua loja (Ex: www.sualoja.com.br)
