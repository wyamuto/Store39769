As lojas de cada idioma são independentes, com preços, descrições de produtos, formas de pagamento, etc específicos.

É possível sincronizar os estoques de cada loja, desde que as referências dos produtos nas duas lojas sejam idênticas.

Ao criar uma loja no Fastcommerce, selecione o idioma da loja no campo Idioma na página Dados da loja do site administrativo.

É possível migrar os produtos de uma loja para outra através de arquivo CSV. Para isto, basta exportar CSV de produtos na loja em português, traduzir o conteúdo e importar na loja com o outro idioma.

Opcionalmente, utilize o recurso de sincronismo de estoques entre as lojas.

O sincronismo de estoques é utilizado para garantir consistência entre lojas de diferentes idiomas e com estoque único, ou para integrar uma loja de atacado e outra de varejo com estoques comuns.

É possível sincronizar estoques de produtos com mesma referência entre diversas lojas interligadas. Desta forma, sempre que ocorrerem alterações nas quantidades de produtos de uma loja, as demais lojas em sincronia terão quantidades alteradas nos produtos de mesma referência.

Alterações sincronizadas nos estoques de produtos ocorrem automaticamente quando o cliente faz um pedido e quando um pedido é cancelado ou descancelado. Alterações sincronizadas também ocorrem na página de cadastro de produtos, sempre que as quantidades em estoque são modificadas.

Para utilizar, preencha os campos Sincronismo Loja na página Dados da loja das lojas interligadas.
