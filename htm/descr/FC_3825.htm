Ao clicar nesta opção, uma janela será exibida com os dados para impressão da nota fiscal deste pedido.

Para utilizar este recurso, é necessário possuir o Microsoft Office instalado em seu micro.

Os dados para a nota fiscal são preparados para uso com o recurso de mala direta do Word, em que dois arquivos são utilizados: um com os dados da nota fiscal (criado pelo sistema) e outro com as posições de impressão exata de cada campo no formulário de notas fiscais da loja (criado pela loja).

Dois formatos de dados estão disponíveis: DOC ou XLS.

Clique com o botão direito sobre a janela e salve este arquivo com o nome sugerido Dados.DOC ou Dados.XLS, sempre na mesma pasta em seu micro.

Em seguida, utilizando o recurso de mala direta, combine este arquivo com o outro DOC contendo a máscara de sua nota fiscal.

Por fim, imprima a nota fiscal completa em sua impressora.

Obs1: O sistema faz o cálculo automático de ICMS para cada produto, utilizando o percentual informado no cadastro do produto.

Obs2: Os campos do arquivo de dados DOC são divididos pelo caractere {.

Obs3: No formato DOC, existe uma limitação de 63 colunas (campos) por tabela, causando um limite de até 10 produtos por fatura. Além disso, algumas informações tiveram de ser agrupadas em um mesmo campo.

Obs4: Também é possível adquirir sistemas de terceiros que estão integrados ao Fastcommerce e que façam a geração de notas fiscais eletrônicas. Um exemplo são os sistemas de ERP integrados ao Fastcommerce em tempo real via API.

Veja informações detalhadas em Integração via API.
