Todas as alterações são feitas pelo site administrativo (sem FTP).

As páginas do site são compostas por cabeçalho + rodapé + barra direita + barra esquerda, além do corpo da página.

Cada parte deve ser enviada em arquivo htm separado.

O Fastcommerce se encarrega de agrupá-las para gerar as páginas.

Você pode utilizar qualquer tag html + javascript + dhtml em seus projetos, além das tags especiais do cHTML: https://www.rumo.com.br/sistema/adm/TagsEspeciais.htm

Veja mais detalhes na página Textos & imagens do site administrativo.

As fontes são definidas através de CSS, desde o tamanho de letra para os preços e textos descritivos de produtos até as cores dos formulários de cadastramento.

O Gerenciador de Arquivos é bem mais seguro e funcional do que o FTP, permitindo o envio de arquivos zipados, que serão automaticamente descompactados no servidor e vice-versa.
