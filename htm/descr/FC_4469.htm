Clique na interrogação à direita deste campo para verificar os detalhes de cada layout pré-definido.

A coluna Tabela parcelas indica se o layout apresenta as tabelas de parcelamento de cada produto.

Outra opção é alterar o arquivo JurosParcelas.js que fica na pasta htm no gerenciador de arquivos de sua loja.

Por exemplo se sua loja não trabalha com parcelamento o arquivo JurosParcelas.js deve ser assim:

//Informe abaixo os juros para parcelamento em 1x, 2x, 3x, etc.
var Juros=new Array(0);
