Em seu arquivo Home.htm para exibir os produtos você deverá utilizar a tag

Ex.:

<prod out="p10" em="rand">

p10 é o modelo exemplo contido no arquivo EstiloProdutoOut.htm.


2) Enviar o arquivo EstiloProdutoOut.htm para a pasta HTM da loja.

Neste modelo em anexo tem uma cópia do modelo OUT=10 com a função abaixo, para exibir parcelamentos.

<script>MostraMaxParcela(<PrecoNum>,<MaxParcelasProdNum>);</script>

Você pode criar outros modelos de exibição de produtos.

function MostraMaxParcela(PrecoProd,MaxParcelas){
  var ComSem;
  if(PrecoProd==0||MaxParcelas==1||Juros.length==0)return;
  if(MaxParcelas==0||MaxParcelas>Juros.length)MaxParcelas=Juros.length;
  if(Juros[MaxParcelas-1]>0)ComSem=""; else ComSem="<font color=#990000> sem juros</font>";
  document.write("ou <b>"+MaxParcelas+"x</b>"+ComSem+" de <b>"+FormatPrecoReais(CalculaParcelaJurosCompostos(PrecoProd,MaxParcelas))+"</b>");
}
</script>
