Ao criar esta imagem não se esqueça que o ícone pode ser visto com diferentes cores de fundo, então é aconselhado que faça a imagem com o fundo transparente ou branco.

Siga os passos abaixo para colocar esta imagem em sua loja:

1) Crie um arquivo .ico (favicon.ico). Clique aqui para converter arquivos de outros formatos para .ico.

2) Envie este arquivo favicon.ico para a pasta images do gerenciador de arquivos da lojav

3) No campo Header HTML adicional da página Dados da loja do site administrativo, insira a linha abaixo:

<link rel="shortcut icon" href="/lojas/00009999/images/favicon.ico">

Altere de 00009999 para o ID da loja com 8 posiçoes, por exemplo, se o ID de sua loja é 123 coloque 00000123.

Caso não consiga visualizar o favicon no Internet Explorer, adicione-o como um favorito.

Obs: Caso queira, também é possível adicionar um ícone para ser exibido em iPhone, iPod e iPad.

Siga os passos abaixo para colocar esta imagem em sua loja:

1) Crie um arquivo .png (FavIconApple.png) com a dimensão 152x152

2) Envie este arquivo FavIconApple.png para a pasta images do gerenciador de arquivos da loja

3) No campo Header HTML adicional da página Dados da loja do site administrativo, insira a linha abaixo:

<link rel="apple-touch-icon" sizes="152x152" href="/lojas/<00009999/images/FavIconApple.png">

Altere de 00009999 para o ID da loja com 8 posiçoes, por exemplo, se o ID de sua loja é 123 coloque 00000123.
