O código html do layout "p1" fica entre as tags <p1> e </p1>.

Para utilizá-los nos demais arquivos HTM da loja, utilize a tag <prod> com a cláusula out=pn, onde n é 1.

Exemplo:

<prod out="p1" em="prom">
