Para busca com os campos de descritores especiais, Adicional1, Adicional2 e Adicional3, informe o ID do item, como no exemplo acima e o parâmetro Avancada=True.

Para buscar pelos descritores simples, utilize o parâmetro Texto, que faz busca nestes campos.

Os descritores especiais permitem predefinir valores que facilitam o cadastramento de autor, genero, marca, tamanhos, cores, etc. É possível cadastrar 3 descritores simples, 3 descritores especiais e cor, na página Dados da loja no site administrativo.
