XML123

Este termo será invisível aos visitantes da loja mas poderá ser usado para restringir os produtos exibidos no XML.

Seguem abaixo as URLs para gerar XML contendo somente estes produtos:

Loja com o serviço de domínio próprio contratado no Fastcommerce:

http://www.minhaloja.com.br/XMLProdutos.asp?IDLoja=99999&texto=XML123

Loja sem o serviço de domínio próprio contratado:

http://minhaloja.lojavirtualfc.com.br/xmlprodutos.asp?IDLoja=99999&texto=XML123

Altere onde está 99999 para o ID de sua loja e altere o nome de minhaloja para o que foi preenchido no campo Endereço Internet em Navegação da página Dados da loja do site administrativo.

Caso a loja tenha o serviço de domínio próprio contratado no Fastcommerce, você deverá alterar o domínio minhaloja.com.br para o domínio de sua loja (Ex: www.sualoja.com.br).
