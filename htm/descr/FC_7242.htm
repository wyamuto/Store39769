Ex: http://www.rumo.com.br/sistema/ListaProdutos.asp?IDLoja=288&IDProduto=1185104


Alterações necessárias:

1ª No arquivo EstiloProduto.htm entre as tags <ProdTop> e </ProdTop> insira a seguinte função:

<script>
function IndicaProd(IDProduto){
 document.write("<center><span class=smText><a href='javascript:MostraIndique(99999,"+IDProduto+")'><img src=/lojas/00007777/images/cartao.gif border=0 vspace=2><br>Recomende este produto</a></span></center>");}
</script>

No exemplo acima existe uma imagem chamada cartao.gif, que deve ser enviada para a pasta IMAGES do gerenciador de arquivos da loja e um texto Recomende este produto, que pode ser alterado de acordo com sua preferência (ex: Quero este produto de presente).

Caso não queira inserir uma imagem, segue outro exemplo:

<script>
function IndicaProd(IDProduto){document.write("<a href='javascript:MostraIndique(99999,"+IDProduto+")'>Recomende</a>");
}
</script>

Altere onde está 99999 para o ID de sua loja (ex. 123) e onde está 00007777 para o ID da loja com 8 posiçoes, por exemplo, se o ID de sua loja for 123 coloque 00000123. Não esqueça de remover a quebra de linha dos scripts.


2ª Remova as tags <IndicaProd> do arquivo EstiloProduto.htm e insira no lugar uma chamada para a função:

<script>IndicaProd(<IDProduto>);</script>

3ª Altere o campo Layout das páginas de produtos da página Dimensões & menus do site administrativo para Personalizado.
