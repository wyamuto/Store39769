Se o arquivo "LojaLib.js" não existir na pasta HTM, adicione o código no começo do arquivo "BarraTopo.htm", conforme exemplos abaixo.

ATENÇÃO: A implantação deste recurso requer CONHECIMENTOS AVANÇADOS de html e JavaScript.

Segue código para ser adicionado em seu arquivo LojaLib.js:

function MostraEconomia(PrecoProd,PrecoOri){ 

  if(PrecoProd!=PrecoOri)document.write("<br><font color=#6f9e45>Economize <b>"+FormatPrice(PrecoOri-PrecoProd,'R$')+"</b> ("+FormatNum(((PrecoOri-PrecoProd)/PrecoOri)*100)+"%)</font>");

}



function FormatNum(num){

  num=num.toString().replace(/\$|\,/g,'');

  if(isNaN(num))num="0";

  sign=(num==(num=Math.abs(num)));

  num=Math.floor(num*100+0.50000000001);

  num=Math.floor(num/100).toString();

  for(var i=0;i<Math.floor((num.length-(1+i))/3);i++)num=num.substring(0,num.length-(4*i+3))+'.'+num.substring(num.length-(4*i+3));

  return ((sign)?'':'-')+num;

}

A chamada para a função deverá ser adicionada no arquivo onde deseja mostrar a informação Economize.

Ex.: Para mostrar na página inicial da loja, adicione a chamada no arquivo EstiloProdutoOut.htm e para mostrar na listagem de produtos, adicione no arquivo EstiloProduto.htm.
