Para incluir linhas neste arquivo, clique no link Arquivos que fica no topo do site administrativo, selecione a pasta HTM e clique na opção "Enviar arquivos" para enviar o arquivo com as linhas adicionais. Este arquivo deve ser enviado com o nome robots.htm

Caso a loja não tenha este arquivo na pasta HTM, um arquivo robots.txt padrão será exibido. Veja exemplo deste arquivo padrão:

http://www.perfumes.com.br/robots.txt

É possível também utilizar a meta tag robots, que pode ser incluida no campo de Header HTML adicional da página Dados da loja, junto com as demais meta tags que podem ser utilizadas para controlar a indexação do site.
