1º Passo

Colocar a tag <FormProdMult> entre <ProdTop> e </ProdTop>


2º Passo

Colocar a tag </FormProdMult> dentro de <ProdBottom> e </ProdBottom>


3º Passo

Colocar tag <IncMult> na seções <ProdLista>, <ProdDet> e <ProdAmp> (e dentro de <Sub>...</Sub> caso tenha sub-produtos).

Esta tag conterá uma caixa de checkbox e um campo para digitar a quantidade de produtos.


4º Passo

Remover as tags <FormProd> e </FormProd>


5º Passo

A tag <LinkComprar> pode ser colocada em cada produto/sub-produto e/ou ao final da página, dentro de <ProdBottom> e </ProdBottom>.

Esta tag conterá o link para adicionar ao carrinho todos produtos selecionados e respectivas quantidades.


6º Passo (opcional)

É possível incluir funções para marcar e desmarcar todos os produtos exibidos na página. Veja exemplo abaixo:

<script>
function checkAll(){
 for (i=0;i<document.FormMult.elements.length;i++)
 {
 if(document.FormMult.elements[i].type=="checkbox")
       document.FormMult.elements[i].checked=1;
 else if(document.FormMult.elements[i].type=="text" &&
document.FormMult.elements[i].value<2)
       document.FormMult.elements[i].value=1;
 }
}

function uncheckAll(){
 for (i=0;i<document.FormMult.elements.length;i++)
 {
 if(document.FormMult.elements[i].type=="checkbox")
       document.FormMult.elements[i].checked=0;
 else if(document.FormMult.elements[i].type=="text")
       document.FormMult.elements[i].value=0;
 }
}

j=0;
for (i=0;i<document.FormMult.elements.length;i++){
 if(document.FormMult.elements[i].type=="checkbox")j=j+1;
}

if(j>1)document.write("<a href='javascript:checkAll()'>Marcar 
todos</a> | <a href='javascript:uncheckAll()'>Desmarcar todos</a>")
</script>
