As tag clouds podem ser úteis tanto para os visitantes quanto para os robôs de indexação dos sites de busca (ex: Google).

Veja imagem com exemplo de Tag Cloud

Clique no botão Gerar ao lado direito do campo Tag cloud na página Dados da loja do site administrativo para que o arquivo TagCloud.htm seja gravado na pasta HTM da loja, contendo os 100 termos mais procurados pelos visitantes da loja, no layout de tag cloud. Para prever o resultado da nuvem se esta fosse colocada no corpo da loja, clique no botão site. Clique no botão htm para ver o código-fonte de TagCloud.htm, que poderá ser editado externamente e reenviado através da seta ao lado deste campo.

Para incluir a nuvem na loja, insira o código do arquivo TagCloud.htm na página desejada ou utilize a tag <TagCloud>, que automaticamente será substituída pelo conteúdo deste arquivo. Antes de incluir, verifique se existem termos indesejados no arquivo TagCloud.htm e remova-os.

ATENÇÃO: Para este recurso funcionar corretamente é necessário que seu campo de busca na loja tenha um input com o parâmetro digitada=true (<input type=hidden name=Digitada value=True>).


Obs¹: A tag cloud criada pelo botão Gerar conterá um mínimo de 20 termos e um máximo de 100 termos. Por esta razão, lojas recentes não conseguirão utilizar esta funcionalidade.

Obs²: Para que um termo seja incluído na tag cloud, pelo menos 3 buscas deverão ter sido feitas com este termo a mais de 24h e cada busca terá que encontrar pelo menos um produto.

Obs³: Para que mantenha sua relevância, periodicamente a tag cloud deverá ser recriada através do botão Gerar.
