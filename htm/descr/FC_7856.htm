Seguem abaixo todas as opções de ordenação:

  1. Padrão

  2. Lançamentos

  3. Destaques

  4. Nomes das categorias

  5. Nomes dos produtos

  6. Avaliações dos clientes

  7. Promoções

  8. Preços menores

  9. Preços maiores


Para alterar o nome destas opções ou para remover alguma delas, inclua o código abaixo ao final do arquivo Rodape.htm:

<script>
var sPag=document.location.href.toUpperCase();
if(sPag.indexOf("BUSCAAVANCADA.ASP")>0){
  var oOrder=document.getElementsByName('order')[0];
  oOrder.options[2].text='Mais vendidos'; //altera o nome da opção "Destaques"
  oOrder.options[5]=null; //remove a opção "Avaliações dos clientes"
}
</script>

Se necessário, altere o código acima para exibir ou alterar as opções desejadas. Os valores padronizados de oOrder.options estão listados na lista acima, à esquerda de cada opção de ordenação.

No exemplo acima, a opção Destaques será substituída pelo texto Mais vendidos. Neste caso, execute periodicamente o utilitário "Definição de produtos mais vendidos" para que os produtos mais vendidos sejma marcados como "Destaque" e a ordenação priorize de fato estes produtos.
