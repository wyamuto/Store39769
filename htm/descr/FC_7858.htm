O navegador Apple Safari é o que mais se assemelha ao navegador web do iPhone e do iPod Touch. O Safari inclui recurso para utilizar o User Agent destes aparelhos.

Para ver a loja pelo Safari deve-se instalar este navegador. Acesse http://www.apple.com/safari para ir ao site da Apple e baixar o arquivo de instalação. Em seguida, abra o Safari, entre no menu Edit e clique em Preferences....


Na janela que se abriu, clique no botão Advanced, marque o campo Show Develop menu in menu bar e feche esta janela.


Ao realizar este procedimento será adicionado no menu do Safari mais um item chamado Develop. Clique neste item, selecione User Agent e marque a opção Mobile Safari n.n.n - iPhone.


Agora basta navegar em sua loja para ver como será a navegação por este dispositivo.
