O campo de sub-categorias fica na página Dados da loja, logo abaixo do campo Aviso de vencimento:


Ao marcá-lo, será possível fazer o cadastro de sub-categorias na ficha de cada categoria e mostrar esta hierarquia colocando as tags especiais <ListPathCat> e <PathCat> nos arquivos HTM personalizados.

Por exemplo, se a loja possui um menu esquerdo com a lista de categorias simples, para que seja exibida uma lista hierárquica com as categorias e sub-categorias, deve-se trocar no arquivo BarraEsquerda.htm a tag <ListCat out="1"> pela tag <ListPathCat out="2"> que mostra menu em HTML. Para que o menu venha com as sub-categorias abertas, utilize o parâmetro id, exemplo: <ListPathCat out="2" id="-5"> que mostra menu em HTML aberto até o último sub-nível, que é o 5º.


Veja mais detalhes sobre as configurações destas tags especiais em:

http://www.rumo.com.br/sistema/adm/TagsEspeciais.htm


Obs: O limite é de até 5 sub-níveis, abaixo da categoria principal.

Atenção: Para otimizar a performance da loja, mantenha este campo desmarcado se a hierarquia de sub-categorias for feita somente no design da loja e não no seu cadastro.
