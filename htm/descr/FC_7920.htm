A contabilização por pageviews tem várias vantagens para o lojista:

  • Mais fácil de entender e de acompanhar.

  • Independe do tamanho de cada página.

  • Permite acompanhar o tráfego com precisão.


Seguem sugestões de relatórios que podem ser utilizados para acompanhar a visitação da loja:

- Acessos mensais no período

Informe nos campos De e Até a data inicial e final que deseja ver no resultado do relatório. Esta data deve ser a data base da loja.


No exemplo acima, a loja teve 21183 pageviews no período informado.


- Visitantes no período

Informe nos campos De e Até a data inicial e final que deseja ver no resultado detalhado do relatório, esta data deve ser a data base da loja. Caso tenha algum problema na geração deste relatório sugerimos que informe períodos menores.


Neste relatório pode-se ver a Sessão, Pedido, Origem, Entrada, Saída, IP, Pageviews e User Agent de cada visita.


- Principais configurações de sistema dos visitantes

Este relatório relaciona os principais navegadores e sistemas operacionais utilizados pelos visitantes. O user-agent pode ser utilizado como filtro no firewall personalizado da loja, para bloquear visitas indesejadas.
