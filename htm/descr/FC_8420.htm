Os arquivos devem ter os seguintes nomes, para cada novo campo:

  • CliCampo1.htm, CliCampo2.htm e CliCampo3.htm

  • PedCampo1.htm, PedCampo2.htm e PedCampo3.htm


Os arquivos CliCampoX.htm são referentes aos campos que vão aparecer na ficha de cadastro dos clientes.

Os arquivos PedCampoX.htm são referentes aos campos que vão aparecer na ficha de dados de entrega do pedido.

Veja recurso SEM personalização do campo:

Veja recurso COM personalização do campo:

Além da personalização é possível validar estes campos, onde o name deverá ser sempre P2CliCampoX ou P2PedCampoX, onde:


  • P2CliCampoX deve ser utilizado quando os campos forem adicionados para o cadastro de cliente.
    Ex.: Se criou o arquivo CliCampo1.htm o "name" deverá ser P2CliCampo1

  • P2PedCampoX deve ser utilizado quando os campos forem adicionados para o dados de entrega.
    Ex.: Se criou o arquivo PedCampo3.htm o "name" deverá ser P2PedCampo3

    Achei=false;
    iArray=0;

      if(document.Form1.P2CliCampoX[iSelect].value==aValores[iArray]){document.Form1.P2CliCampoX[iSelect].checked=true;Achei=true;}
      else iArray++;
    }
    iSelect++;
  }
}
