Clique aqui para baixar o arquivo EstiloProduto.htm com o layout semelhante ao do exemplo acima.

Obs.: Para este recurso funcionar corretamente, deve ser cadastrado um sub-produto para cada tamanho.


Veja o passo-a-passo para fazer esta personalização:


1º Passo

Na página Dados da loja do site administrativo, escreva Tamanho no campo Descritor especial 3 e clique na seta ao lado direito deste campo para cadastrar os tamanhos que a loja oferecerá (ex.: P, M, G, 35, 36, 37, etc).

Obs: Se utilizar outro campo de descritor especial 1 ou 2, será necessário alterações no código html do arquivo EstiloProduto.htm.


2º Passo

Cadastre o produto PAI. No campo Estoque preencha 1. Informe o preço normal e/ou promocional, as datas de início de término da promoção. NÃO informe tamanhos para este produto.


3º Passo

No cadastro do sub-produto, informe apenas o estoque, o peso, o preço (que deve ser o mesmo do produto PAI) e selecione apenas um tamanho.

Repita o passo 3 para cada tamanho do produto.


4º Passo

Envie o arquivo EstiloProduto.htm para a pasta htm de sua loja.


ATENÇÃO: Antes de realizar alterações em arquivos de imagens e html, sugerimos que efetue backup de seus arquivos para reverter as alterações se necessário.
