Veja mais informações sobre o QR Code em:

http://pt.wikipedia.org/wiki/QR_Code

O conteúdo deste QR Code é o link da página que está sendo visitada.

Para adicionar na sua loja, envie o arquivo QRCode.js para a pasta htm. Clique aqui para baixar este arquivo.

Copie todo o código abaixo e adicione no arquivo htm de acordo com a local desejado para o código (exemplo: rodape.htm):
