1º passo

Cadastre o campo adicional, clicando na seta ao lado direito do campo na página Dados da loja do site administrativo.

Na janela Campos adicionais de clientes, informe Redigite seu e-mail no campo Campo 1 da tabela Cadastro do cliente e selecione no campo 50 no campo Máx.


2º passo

Copie o código abaixo ao final do arquivo LojaSite.css, para que o novo campo fique com a cor dos campos obrigatórios.

#idCliCampo1 .EstNomeCampo {color:#ff2200}

Neste exemplo utilizamos a cor vermelha. Caso utilize outra cor, altere o código ff2200.


3º passo

Crie um arquivo chamado CliCampo1.htm com o conteúdo abaixo e envie este arquivo para a pasta htm da loja.

<input class=InputText type=text name=P2CliCampo1 id=P2CliCampo1 maxlength=50 style=width:300px>
<script>function CliValidaCampo1(){
  if(document.Form1.P2CliCampo1.value==""){
    alert('Redigite seu e-mail.');
    document.Form1.P2CliCampo1.focus();
    return false;
  }
  else if(document.Form1.P2CliCampo1.value!=document.Form1.P2Email.value){
    alert('E-mail e confirmação diferem.');
    document.Form1.P2CliCampo1.focus();
    }
  else{return true;}
}
</script>

ATENÇÃO: Este recurso está programado para funcionar no Campo 1. Se for utilizar outro campo, será necessário alterações em todo o código acima.
