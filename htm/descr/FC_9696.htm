Segue abaixo exemplo de script:

<script type="text/javascript">LinkDisp(<Estoque>,<IDProduto>);</script>

Também no arquivo EstiloProduto.htm, insira o script abaixo entre as tags <ProdTop> e </ProdTop>:

<script type="text/javascript">function LinkDisp(Estoque,IDProduto){
 if(Estoque==0){
    document.write("<a href='javascript:MostraDisp(99999,"+ IDProduto +")'
    title='Clique aqui para ser avisado quando este produto estiver disponível'>
    Avise-me quando estiver disponível.</a>");
 }
}
</script>

No script acima, altere o ID da loja de 99999 para o ID de sua loja, e o texto padrão para o texto personalizado ou para uma imagem.
