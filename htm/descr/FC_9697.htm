1º Motivo

Faltam as tags <FormProd> e </FormProd> dentro de <ProdLista>...</ProdLista> e/ou <ProdDet>...</ProdDet> e/ou <ProdAmp>...</ProdAmp>.

Veja mais detalhes sobre estas tags em

http://www.rumo.com.br/sistema/adm/TagsEspeciaisProdutos.htm


2º Motivo

Na personalização que foi feita para exibir o produto na loja no arquivo EstiloProduto.htm, não foram inseridas tags para que exibir os descritores.

Se for apenas uma opção no descritor, o produto pode ser adicionado ao carrinho. Mas quando o descritor é multi-valorado (exemplo: escolha de tamanho ou cor do produto), o cliente sempre terá que escolher uma opção.

Para corrigir, coloque a tag de descritor no arquivo EstiloProduto.htm ou deixe apenas com um valor nos descritores na ficha de cada produto.

Verifique também se as tags dos descritores estão dentro das tags <FormProd> e </FormProd>.
