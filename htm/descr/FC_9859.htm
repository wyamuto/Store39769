Opção 1

Se a loja utilizar sub-produtos, faça o cadastro da seguinte maneira para que não seja exibido o link Consulte-nos nos produtos principais:


  1. Somente os sub-produtos serão vendidos. Assim, será necessário cadastrar pelo menos um sub-produto por produto.

  2. Todos os produtos PAI deverão receber preços (mesmo que tenha que repetir o preço dos sub-produtos).


Feito isso, no arquivo EstiloProduto.htm deve-se apagar a tag <LinkComprar> e <Preco> dentro do <ProdDet>...</ProdDet> e <ProdAmp>...</ProdAmp>.

Dentro das tags <ProdDet>...</ProdDet> e <ProdAmp>...</ProdAmp>, existem as tags <SUB>. Não apague as tags <LinkComprar> e <Preco> que estão dentro das <SUB>.


Opção 2

Fazer uma personalização na lista/detalhes dos produtos:

Esta personalização é feita através do arquivo EstiloProduto.htm, campo Layout das páginas de produtos da página Dimensões & menus do site administrativo.

Normalmente a tag <Preco> exibe o preço do produto, ou quando este for igual a zero, o link Consulte-nos. Para mudar este comportamento, será necessário programar neste arquivo um javascript para só exibir o preço se este não for zero. Para isto, utilize a tag especial <PrecoNum>. Segue exemplo:

<script>if(<PrecoNum>>0){document.write("<Preco>");}</script>

Utilize este script no lugar da tag <Preco> no arquivo EstiloProduto.htm. Desta forma, se o preço for zero, não será exibido nada.
