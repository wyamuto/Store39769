1) Choose the domain name

When choosing the domain, prefer short names with easy typing. Avoid numbers and names that are hard to spell or tough to pronounce.

Choose a name that is easy to remember. Also try to use words that describe your company so visitors know what your business is about. This can be helpful for search rankings.

Avoid names that are similar to existing brands.

2) Choose the domain suffix (e.g. .com, .org, .net)

The suffix identifies the name as belonging to a specific top-level domain (TLD). There are numerous TLDs available for general purchase, including .com, .org and .net. The most popular is .com, which is supposed to indicate commercial sites, but includes almost everything.

3) Choose a domain registrar

You must choose a registrar to host your new domain. There are many companies that offer this service, like GoDaddy and Verisign. Please choose only an ICANN accredited registrar.

After registering your domain, you will need to create two DNS entries in the registrar's control panel:

  1. Log into your domain registrar's web portal.
  2. Launch the DNS Manager / Domain Management Page / DNS Control Panel for your account, which lists all the domains you have already registered.
  3. Locate the registered domain you will use for your store.
  4. Locate the CNAME WWW record for this domain.
  5. Remove any existing entries that you might have in this record.
  6. Select the Add Record option.
  7. Specify the value in Name / Host field as www.
  8. Specify the first record's value as buzios.rumo.com.br.
  9. Click Save zone file/ Save / Add records to save your changes.
  10. Select the Add Record option.
  11. Specify the value in Name / Host field as either @ or leave it blank.
  12. Specify the first record's value as 201.20.10.204.
  13. Click Save zone file / Save / Add records to save your changes.

4) Wait the domain to propagate

After you register and configure the domain name, you will have to wait some hours or even some days for it to propagate. Typically, you should expect to see the domain name up and running within 48 hours.
