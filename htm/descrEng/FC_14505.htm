This article shows you how to add a button to share your store's product in WhatsApp.

The visitor who clicks the button will see a list of their contacts and groups in WhatsApp. You can then select a contact or group that will receive a message through WhatsApp containing a link to the product page.

It is recommended to insert this button into the mobile version the virtual store or in your responsive version, when the customer is in the design made for smartphones.

Button example in a responsive design

Example of message sent to WhatsApp


Here is an example code to insert the button :

<span class="ShareWhatsApp"><a href="whatsapp://send?text=See the product '<NomeProd>'. Follow the link: http://lvfc.co/l<IDLojaNum>/p<IDProduto>" target="_blank"><img src="<BaseLoja>images/IcWhatsApp.svg" width="35" height="35" border="0" alt="Recommend this product by WhatsApp" title="Recommend this product by WhatsApp"></a></span>

If the store has activated the mobile version of the virtual store, this code must be inserted in the file MbhEstiloProduto.htm, within the section <ProdDet>...</ProdDet>, which is where the detail design of the product is.

To enter in the responsive version of the virtual store, this code must be inserted in the file EstiloProduto.htm, within the section <ProdDet>...</ProdDet>, which is where the detail design of the product is. In addition to entering this code, the store must insert in the file LojaSite.css the class ShareWhatsApp to make the button appear only in the design displayed on the mobile phone.


Obs1: The image IcWhatsApp.svg must be sent to the folder Images by the store's file manager. click here to download this image.

Obs2: The position where the code should be inserted will depend on where you want the button to appear.
