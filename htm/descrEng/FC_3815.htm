These utilities create files .csv in the folder HTM from the store's file manager.

To copy the files from the store, such as photos, HTML, CSS, images, script, etc., into the administrative site, enter the Files item (upper right corner), then select the desired folder.

Use the option to mark everyone at the bottom of the page and select the item Download checked to the end of the page to receive the ZIP file with the files.


ATTENTION: To optimize store performance, we ask you to delete the file .CSV of the folder htm soon after its generation and receipt.
