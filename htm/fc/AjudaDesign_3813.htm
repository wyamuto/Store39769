3) Tags especiais para criar links, caminhos para imagens, etc

Veja exemplos de como montar links internos para os produtos, categorias, contato, etc:
http://www.rumo.com.br/sistema/adm/ExemplosLinks.htm


Você pode utilizar qualquer tag html + javascript + java + dhtml em seus projetos, além das tags especiais do cHTML:
http://www.rumo.com.br/sistema/adm/TagsEspeciais.htm


Exemplo 1 : Para criar link para uma página chamada QuemSomos.htm, utilize:

<a href="<HTMLoja>QuemSomos.htm">Quem somos</a>

ou

<a href="Custom.asp?IDLoja=<IDLoja>&arq=QuemSomos.htm">Quem somos</a>

com URL amigável:

<a href="/page,idloja,<IDLojaNum>,arq,QuemSomos.htm,sobre-a-nossa-empresa">Quem somos</a>



Exemplo 2 : Para criar link para uma imagem chamada Imagemdaloja.jpg, utilize:

<img src="<BaseLoja>images/Imagemdaloja.jpg">


A tag <Baseloja> é substituída automaticamente pelo sistema por ../lojas/00000000/, desta forma o caminho para a imagem fica ../lojas/00000000/images/Imagemdaloja.jpg. (00000000 é o nome da pasta correspondente ao ID da loja com 8 posições. Se o ID da loja for 154, ficaria 00000154)




4) Títulos, botões e fundo da loja

Os títulos, botões e fundo da loja são definidos na página Estilos & cores do site administrativo, onde é possível também definir o tipo das imagens (GIF, JPG ou PNG)

Para ver o nome dos arquivos de títulos e ícones de navegação, clique no primeiro botão Visualizar da página Estilos & cores, no site administrativo. Clique aqui para ver exemplo.

Alguns destes erros causam a exibição de alerta de segurança em páginas acessadas com https:. Outros impedem a exibição de imagens ou prejudicam a performance da navegação. Tais erros podem ter consequências negativas nos resultados financeiros da loja.
