1) Como começar

Caso você não tenha conta no MercadoLivre, clique aqui para criá-la.



2) Ativando a integração

A configuração da integração é feita na página Dados da loja do site administrativo, na seção MercadoLivre. Para ativar a integração, selecione o país e clique no botão Autorizar integração, que abrirá uma janela para digitar o apelido e a senha de seu usuário no MercadoLivre. Se esta janela não abrir, desative o recurso de anti-popup de seu navegador. Em seguida, marque o campo Ativo e grave.

    Sincronismo    
Opcionalmente, altere os campos Sincronismo de estoque e Sincronismo de preço para indicar como serão feitos estes sincronismos por padrão:
  • Não: nenhum sincronismo.
  • Loja -> MercadoLivre: alterações feitas na loja são refletidas no MercadoLivre.
  • Loja <- MercadoLivre: alterações feitas no MercadoLivre são refletidas na loja.
  • Loja <-> MercadoLivre: alterações feitas na loja são enviadas para o MercadoLivre e vice-versa.


    Recadastro automático    
Marque o campo Recadastro automático para que os anúncios sejam republicados quando forem finalizados.


    Padronizando os títulos dos anúncios    
Preencha os campos Título padrão dos anúncios e Subtítulo padrão dos anúncios com o texto que será utilizado por padrão na criação de novos anúncios. Nestes campos pode-se utilizar tags especiais, que serão substituídas por informações dos produtos.


    Modelo de descrição (template)    
Opcionalmente, informe no campo Modelo HTM de descrição o nome do arquivo HTML com o design do anúncio. Clique no ícone que fica ao lado do campo para enviar este arquivo para a pasta HTM da loja.

Clique aqui para ver lista de tags especiais que podem ser utilizadas na descrição dos anúncios.


    Modelo de descrição (template)    
Clique no botão Categorias favoritas para navegar pela hierarquia de categorias do MercadoLivre e selecionar aquelas que serão associadas a categorias e anúncios da loja.


    Importando anúncios do MercadoLivre    
Clique no botão Importar anúncios para trazer para a loja virtual os anúncios pré-existentes do usuário no MercadoLivre. Após a importação, será possível associar estes anúncios a produtos da loja, através da ficha de cada produto.



3) Importando anúncios do MercadoLivre para a loja virtual
Caso já venda através do MercadoLivre, utilize esta opção para trazer para o site administrativo da loja os anúncios do usuário no MercadoLivre. Após a importação, será possível associar estes anúncios a produtos da loja, através da ficha de cada produto.



4) Associando os anúncios aos produtos da loja
Após importar os anúncios do MercadoLivre, é possível associar os anúncios aos produtos da loja e utilizar o recurso de sincronismo se desejar.

Para associar, acesse a ficha de cadastro de um produto no site administrativo e clique no botão Associar anúncio. Uma lista de anúncios sem produtos associados será exibida com um botão ao lado de cada anúncio para associar:

5) Definindo as categorias do MercadoLivre
Clique no botão Categorias favoritas da seção MercadoLivre da página Dados da loja do site administrativo para navegar pela hierarquia de categorias do MercadoLivre e selecionar aquelas que serão associadas a categorias e anúncios da loja. Estas categorias favoritas irão faciliar o trabalho de criação de anúncios de forma individual e na associação destas categorias às categorias da loja. Esta associação é utilizada para a exportação em massa de produtos para o MercadoLivre.

6) Exportando os produtos da loja virtual para o MercadoLivre
Através deste recurso é possível exportar produtos em massa para o MercadoLivre, criando anúncios associados e, opcionalmente, sincronizados com a loja. Veja um passo-a-passo.

Antes de exportar, é preciso associar uma ou mais categorias da loja às respectivas categorias do MercadoLivre. Esta associação é feita através do campo Categoria no MercadoLivre que fica na ficha de cada catetoria.

A página de exportação é dividida em três seções:
  • Filtro de produtos: Nesta seção serão informados os critérios de seleção dos produtos que serão exportados.


  • Dados dos novos anúncios: os anúncios exportados serão criados com as informações contidas nesta seção, incluindo título, subtítulo, imagens etc.


  • Descrições dos anúncios: o design padronizado dos anúncios exportados é definido nesta seção.



O campo Multiplicador do preço pode ser utilizado para alterar os preços dos produtos exportados. Se este campo for preenchido com valor diferente de 1, os anúncios criados terão preços diferentes dos produtos da loja. Por exemplo, para oferecer um desconto de 30% nos anúncios, preencha este campo com 0,7. Para aumento de 15%, preencha este campo com 1,15.

Marque o campo Recadastro automático para que os anúncios finalizados sejam republicados automaticamente.

A quantidade e a data de início dos anúncios são campos opcionais. Se a quantidade não for informada, será utilizada a quantidade em estoque de cada produto. Se a data de início não for informada, a publicação do anúncio ocorrerá logo após a liberação do mesmo pelo MercadoLivre.

Clique aqui para ver lista de tags especiais que podem ser utilizadas na descrição dos anúncios.

Antes de exportar, selecione o tipo de anúncio de acordo com a respectiva tarifa e a exposição desejada para os anúncios.

8) Criando anúncios de forma individual
É possível criar um anúncio de forma individual no MercadoLivre. Veja detalhes, clicando aqui.

8.1) Anúncios mais atraentes e vendedores:

O design da descrição do produto poderá utilizar modelos prontos ou ser totalmente personalizado. Seu arquivo HTML poderá conter tags especiais, que serão automaticamente substituídas por informações do produto (nome, preço, descrição etc). Clique aqui para mais informações.


8.2) Calculador de frete:

Cada anúncio poderá conter um link onde o próprio cliente obterá o valor do frete, informando seu CEP e a quantidade de itens desejados. Clique aqui para mais informações.



9) Gerenciando os pedidos
Os novos pedidos feitos no MercadoLivre serão automaticamente importados para a loja virtual, para facilitar seu gerenciamento e permitir o uso dos relatórios do Fastcommerce. Seus status na loja serão mantidos sempre atualizados, incluindo os status de pagamento e de envio.

É possível utilizar as centenas de recursos de administração de pedidos de uma loja virtual nos pedidos importados do MercadoLivre. Veja alguns exemplos:
  • Fluxo multi-departamental dos pedidos
  • Envio de mensagens pré-formatadas
  • Diversas formas de pagamento
  • Relatórios administrativos
  • Emissão de nota fiscal eletrônica (NFe)



10) Gerenciando perguntas e respostas
Através do site administrativo, é possível responder às perguntas feitas sobre cada anúncio e administrar as pendências. É possível também criar mensagens pré-formatadas com tags especiais dinâmicas para utilizar nas respostas e desta forma agilizar as respostas. (Site administrativo > Menu Mensagens > Mensagens enviadas para os clientes > Botão Incluir nova mensagem > Tipo: MercadoLivre)
