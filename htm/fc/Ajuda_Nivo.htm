Veja o Nivo Slider em funcionamento, na troca de imagens abaixo:



1º Passo
Envie o arquivo NivoHTM.zip para a pasta htm da loja:
Download: NivoHTM.zip

2º Passo
Envie o arquivo NivoImages.zip para a pasta images da loja:
Download: NivoImages.zip

3º Passo
Adicione as linhas abaixo no inicio do arquivo BarraTopo.htm:
<script type="text/javascript" src="/jquery-1.7.1.min.js"></script>
<script type="text/javascript" src="<BaseLoja>htm/nivo_slider.js"></script>
<link rel="stylesheet" href="<BaseLoja>htm/nivo.css" type="text/css" media="screen" />
<link rel="stylesheet" href="<BaseLoja>htm/nivo_slider.css" type="text/css" media="screen"/>
<script type="text/javascript">
  jQuery.noConflict();
    jQuery(window).load(function() {
      jQuery('#slider').nivoSlider({
    });
  });
</script>

4º Passo
Adicione o código abaixo no arquivo onde deseja inserir o slider. Exemplo: Home.htm

<div id="slider" class="nivoSlider">
   <img src="<BaseLoja>images/XXXXX1.jpg"/>
   <img src="<BaseLoja>images/XXXXX2.jpg"/>
   <img src="<BaseLoja>images/XXXXX3.jpg"/>
   <img src="<BaseLoja>images/XXXXX4.jpg"/>
   <img src="<BaseLoja>images/XXXXX5.jpg"/>
   <img src="<BaseLoja>images/XXXXX6.jpg"/>
</div>

Obs¹: Altere XXXXX1.jpg a XXXXX6.jpg pelos nomes das imagens que irão rotacionar no slide. As imagens devem ser enviadas para a pasta images da loja.

Obs²: Para adicionar um link nas imagens do slider, basta incluir o código abaixo, alterando www.Fastcommerce.com para o endereço desejado:
<a href="http://www.Fastcommerce.com"><img src="<BaseLoja>images/XXXXX.jpg"/></a>
