1º Passo Adicione no começo do arquivo BarraTopo.htm o código abaixo: 
<script>
//Guarda em variável a página atual
var sPagAtual=document.location.href.toUpperCase();
var isPaginaPedido=(sPagAtual.indexOf("FORMAPAGTO.ASP")!=-1 || sPagAtual.indexOf("RECALCULA.ASP")!=-1 || sPagAtual.indexOf("CADASTRO")!=-1)

</script>


2º Passo Segue exemplo de código para ser utilizado no arquivo BarraTopo.htm: 
<div id=LayoutNormal_BarraTopo style="display:none">
  Design do topo normal (até a cesta de produtos).

</div>

<div id=LayoutPedido_BarraTopo style="display:none">
  Design do topo para os passos de finalização do pedido:
  (cálculo de frete, escolha da forma de pagamento, cadastro, confirmação do pedido e pedido confirmado.)

</div>

<script>
if(isPaginaPedido)
{
  document.getElementById("LayoutNormal_BarraTopo").style.display='none';
  document.getElementById("LayoutPedido_BarraTopo").style.display='';

}
else 
{
  document.getElementById("LayoutPedido_BarraTopo").style.display='none';
  document.getElementById("LayoutNormal_BarraTopo").style.display='';

}
</script>
