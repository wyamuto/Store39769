4º Passo
Adicione o código abaixo no local onde existe a tag especial que mostra a imagem do produto no arquivo EstiloProduto.htm entre as tags

Se preferir, clique aqui para baixar um modelo do EstiloProduto.htm com este código.
      <!-- começo :: zoom com várias imagens -->
      <script>
      var imgDetAll = "<ImagemProdDet>";
      var imgAmpAll = "<ImagemProdAmp>";
      var novoArray = imgDetAll.split(',');
      var novoArrayAmp = imgAmpAll.split(',');
      var CountImgDet=novoArray.length;
      var CountImgAmp=novoArrayAmp.length;
      
      if (imgDetAll=="" || imgAmpAll==""){
        document.write("");
      }
      else{
        if (CountImgDet==CountImgAmp){
          var FC_MaxImages=CountImgDet-1;
          MostraZoomImgFC();
        }
        else{
          var FC_MaxImages=0;
          MostraZoomImgFC();
        }
      }
      
      function MostraZoomImgFC(){
        for (var i=0;i<=FC_MaxImages;i++)
        {
          if(i==0)
          {
            imgDetMini=novoArray[i];
            imgAmpMini=novoArrayAmp[i];
            document.write("<a href="+imgAmpMini+" title=<NomeProd> class=MagicZoom id=Zoomer><img src="+ imgDetMini +"></a><br>");
            document.write("<a href="+imgAmpMini+"  rel='zoom-id:Zoomer;' rev="+ imgDetMini +"><img src="+ imgDetMini +" width=75 height=75 class=ZoomIMG></a>");
          }
          else{
                imgDetMini="<BaseLoja>prod/"+novoArray[i];
                imgAmpMini="<BaseLoja>prod/"+novoArrayAmp[i];
                document.write("<a href="+imgAmpMini+" rel='zoom-id:Zoomer;' rev="+ imgDetMini +"><img src="+ imgDetMini +" width=75 height=75 class=ZoomIMG></a>");
          }
        }
      }
      </script>
      <!-- fim :: zoom com várias imagens -->

5º Passo
Todas as imagens que são apresentadas devem ser cadastradas entre vírgulas nos campos Imagem de detalhe e Imagem ampliada na ficha
dos produtos.

Exemplo:
Imagem de detalhe: ImagemDetalhe01.jpg,ImagemDetalhe02.jpg,ImagemDetalhe03.jpg,ImagemDetalhe04.jpg
Imagem ampliada: ImagemAmpliada01.jpg,ImagemAmpliada02.jpg,ImagemAmpliada03.jpg,ImagemAmpliada04.jpg

Obs:Todas as imagens devem ser enviadas para a pasta prod antes de preencher os campos na ficha de produto.


Comportamento previsto:

Se o número de imagens preenchidas na ficha do produto for diferente, só será exibida a primeira imagem.

Exemplo: Cadastradas 3 imagens de detalhe e apenas 2 imagens ampliadas.
