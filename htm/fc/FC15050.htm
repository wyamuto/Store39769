É possível personalizar estes botões da seguinte forma:

Para trocar o formato do botão, insira no arquivo lojalib.js da pasta HTM do gerenciador de arquivos a variável iModWishlist com valor 1 ou 2:

var iModWishlist=1;

var iModWishlist=2;

Para criar um design personalizado, uma função com o nome FuncButtonAddToWL deverá ser inserida no arquivo lojalib.js da pasta HTM do gerenciador de arquivos:

function FuncButtonAddToWL(idp,bAdd){
  var sCont="";
  if(bAdd==true)sCont="<a title='Produto adicionado na lista de desejos. Clique aqui para ir para a lista de desejos.' href=\"/account.asp?idloja="+ FC$.IDLoja +"&wishlist=1#Wishlist\"><img src='"+ FC$.PathImg +"wishlist_on.png'></a>";
  else sCont="<a href=\"#wl\" title='Clique aqui para adicionar o produto a lista de desejos' onclick=\"WL$.fnAddToWishlist("+idp+")\" rel=\"nofollow\"><img src='"+ FC$.PathImg +"wishlist_off.png'></a>";
  return sCont;
}

Lista de variáveis de configuração para colocar no arquivo lojalib.js:

FC$.Wishlist=0;

Desativa a exibição do botão para adicionar na lista de desejos dentro do script da grade de produtos. Útil para quando a loja for utilizar a tag <wishlist> para inserir o botão no produto principal através dos arquivos EstiloProduto.htm (lista e detalhe do produto) e EstiloProdutoOut.htm (capa).

var bUseLinkToSubWishlist=true;

O default é que o link que existe no produto na lista de desejos vá para o produto principal. Caso a loja defina esta variável como true, o link irá direto para o subproduto, caso seja um subproduto e a loja tenha URL personalizada cadastrada em todos os subprodutos.

var bBuyWishlist=false;

O default na lista de desejos é exibir o botão "Comprar". Caso a loja não queira exibir o botão Comprar na lista de desejos, colocar como false.

É útil por exemplo para lojas que não queiram que o cliente coloque subprodutos na lista de desejos, deixando colocar apenas produtos principais. Neste caso, pode usar a tag <wishlist> em produtos na capa da loja, na lista e no detalhe, sempre colocando no produto principal. Dessa forma não haverá o botão Comprar na lista de desejos. O cliente clicará em "+ detalhes" para comprar um dos subprodutos disponíveis.

Opção de colocar produtos da lista de desejos no design da loja

É possível também exibir os produtos que estão na lista de desejos do cliente logado no design da loja, recomendando estes produtos para ele.

Exemplo:

Dentro de fnFooter() coloque a seguinte linha abaixo da linha sF$.fnLoadXMLPageHistory();:

if(FC$.ClientID>0)MyWL$.fnLoadWL(); //exemplo de como inserir os produtos da lista de desejos no design da loja

Neste exemplo os produtos serão exibidos no rodapé da loja (arquivo Rodape.htm). Crie um design e coloque o id idWishlist que é onde será exibido os produtos da lista de desejos:

<!-- Begin: Wishlist-->
<div class="MainContainer">
  <div class="row">
    <div class="col-small-12 col-medium-12 col-large-12 col-xlarge-12 text-center">
      <div id="idWishlist"></div>
    </div>
  </div>
</div>
<!-- End: wishlist -->
