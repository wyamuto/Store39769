Para cada produto, é possível cadastrar até 3 descritores simples, 3 descritores especiais e um descritor especial para cores. Este cadastro é feito na página Dados da loja do site administrativo.
Cores

Para cadastrar novas cores, acesse a página Dados da loja no site administrativo, na seção Produtos:



1º Passo Marque o campo Exibe cores na página Dados da Loja.
2º Passo Clique na seta à direita do campo Exibe cores.



3º Passo Clique na opção [nova cor].
4º Passo Informe o nome da cor.
5º Passo Selecione a cor na paleta de cores.
6º Passo Clique no botão Incluir.
7º Passo Clique no botão Gravar no final da página Dados da loja.
Descritores especiais

Para cadastrar um descritor especial, acesse a página Dados da loja no site administrativo na seção Produtos:



1º Passo Preencha o nome do descritor (exemplo: Tamanho).
2º Passo Clique na seta à direita do campo do nome do descritor para cadastrar os valores.



3º Passo Clique na opção [nova item].
4º Passo Informe o valor do descritor (exemplo: GG).
5º Passo Clique no botão Incluir.
6º Passo Clique no botão Gravar localizado no final da página Dados da Loja.

Obs.:A ordem do cadastro dos tamanhos será a mesma ordem exibida na loja. Por exemplo, para tamanhos cadastre primeiro P, depois M e depois G, para que estes tamanhos sejam exibidos neste ordem, que não é alfabética.

Cadastre todos os tamanhos existentes, mesmo que ainda não tenha produtos com alguns destes tamanhos. Assim, quando for cadastrar um produto de um tamanho novo, este tamanho já estará cadastrado e na ordem correta.

Personalização HTML

Para montar links para um determinado descritor especial (Ex: produtos de um determinado tamanho), utilize o seguinte formato:

<a href="ListaProdutos.asp?IDLoja=<IDLoja>&Avancada=1&Adicional1=000000">Tamanho P</a>

Exemplo utilizando o formato de URL amigável:

<a href="/prod,idloja,16391,avancada,1,adicional1,000000,tamanho-P">Tamanho P</a>

Nos exemplos acima, altere 000000 pelo ID do Tamanho P.

  • adicional1: indica qual é o descritor usado (por exemplo Tamanho)
  • 000000: indica qual é o ID do Tamanho P (exemplo: 988)
  • tamanho-P: indica o nome do descritor para ser exibido na URL
O link acima lista todos os produtos que estão cadastrados com o Tamanho P.
Descritores Simples

Para cadastrar um descritor simples, acesse a página Dados da loja no site administrativo, na seção Produto:



1º Passo Preencha o nome do descritor (exemplo: Volume).
2º Passo Clique no botão Gravar localizado no final da página Dados da Loja

Obs.: O valor do descritor simples deve ser informado na página de cadastro do produto (exemplo: 100ml).
