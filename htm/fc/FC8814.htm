O modelo FCPlayerSmall (.mp3) possui seis modelos de players. A cor de fundo pode ser alterada de acordo com a preferência da loja.
Modelo 1 Modelo 2 Modelo 3 Modelo 4 Modelo 5 Modelo 6

Seguem instruções de implantação deste player:

1º Passo
O arquivo de audio deve ser hospedado em outro servidor, fora do Fastcommerce.

2º Passo
Entre na ficha do produto e informe no campo URL de apoio o caminho para o arquivo de áudio, precedido do caractere #.

Exemplo: #http://www.sitemusic.com.br/meuarquivo.mp3

3º Passo
No arquivo EstiloProduto.htm coloque no lugar da tag <DescrURL> o código abaixo (no exemplo abaixo utilizamos o modelo 1):

<script>
if('<DescrURL>'.charAt(0)=='<')document.write('<DescrURL>');
else if('<DescrURL>'.toLowerCase().indexOf('.mp3')>0)
  FlashSemBorda('FCPlayerSmall1?file=<DescrURL>&cf=003366&l=1',34,34,'ffffff','');
</script>

4º Passo
Para escolher outro modelo, altere onde está 1, para o número do modelo desejado.

5º Passo
Para alterar a legenda que fica abaixo do player, altere o parâmetro &l=1 de acordo com o idioma da loja (1=português brasileiro 2=inglês 3=espanhol 4=português de Portugal). Para não exibir a legenda, remova o parâmetro &l=1.

6º Passo
Para alterar a cor, mude onde está 003366 para o código da cor desejada.

Exemplo: Código da cor vermelha ff0000 com o modelo 5

O modelo FCPlayerFull possui um modelo de player. A cor de fundo pode ser alterada.

Com este player é possível ouvir só o audio de um arquivo .flv ou .swf com vídeo.
Modelo 1

Seguem instruções de implantação deste player:

1º Passo
O arquivo .flv ou .swf deve ser hospedado em outro servidor, fora do Fastcommerce.

2º Passo
Crie um arquivo XML com o caminho onde os arquivos de áudio estão hospedados. Este arquivo XML deverá ser enviado para a pasta PROD da loja. Exemplo: Playlist.xml
<?xml version="1.0" encoding="UTF-8"?>
<musicas>
<musica nomeMusica="Music 1" nomeArtista="Bob" mp3="http://www.sitemusic.com.br/music1.mp3"/>
<musica nomeMusica="Music 2" nomeArtista="Bob" mp3="http://www.sitemusic.com.br/music2.mp3"/>
<musica nomeMusica="Music 3" nomeArtista="Bob" mp3="http://www.sitemusic.com.br/music3.mp3"/>
</musicas>
Onde:
nomeMusica deve ser informado o nome da música
nomeArtista deve ser informado o nome do artista
mp3 deve ser informado o caminho onde o MP3 foi hospedado

3º Passo
Entre na ficha do produto e informe no campo URL de apoio o caminho onde o XML foi hospedado, precedido do caractere #.

Exemplo: #/lojas/99999999/prod/meuarquivo.xml (altere 99999999 pelo ID da loja com 8 dígitos. Ex: 00000154)

4º Passo
No arquivo EstiloProduto.htm coloque no lugar da tag <DescrURL> o código abaixo:

<script>
if('<DescrURL>'.charAt(0)=='<')document.write('<DescrURL>');
else if('<DescrURL>'.toLowerCase().indexOf('.xml')>0)
  FlashSemBorda('FCPlayerFull?file=<DescrURL>&cf=003366',290,140,'ffffff','');
</script>

5º Passo
A cor pode ser alterada mudando, onde está 003366 para o código da cor desejada.

Exemplo: Código da cor vermelha ff0000 com o modelo 2
