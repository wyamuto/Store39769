function MostraMaxParcelaJS(PrecoProd,MaxParcelas){
  var ComSem;

  if(PrecoProd>=300)MaxParcelas=6;
  else if(PrecoProd>=250)MaxParcelas=5;
  else if(PrecoProd>=200)MaxParcelas=4;
  else if(PrecoProd>=150)MaxParcelas=3;
  else if(PrecoProd>=100)MaxParcelas=2;
  else if(PrecoProd>=1)MaxParcelas=1;

  if(PrecoProd==0||MaxParcelas==1||Juros.length==0)return "";
  if(MaxParcelas==0||MaxParcelas>Juros.length)MaxParcelas=Juros.length;
  if(Juros[MaxParcelas-1]>0)ComSem=""; else ComSem="<font color=#990000> sem juros</font>";
  return "<span class=EstParc> ou <b>"+MaxParcelas+"x</b>"+ComSem+" de <b>"+FormatPrecoReais(CalculaParcelaJurosCompostos(PrecoProd,MaxParcelas))+"</b></span>";
}

Como utilizar imagens para as cores?

Cadastre as cores na página de Dados da Loja >> Produtos >> clique na seta a direita do campo "Exibe cores", informe o nome da cor, selecione uma cor na tabela de cores e clique no botão incluir.

Para mais detalhes sobre como cadastrar as cores e demais descritores para os produtos clique aqui.

Crie uma pasta com o nome CORES dentro da pasta PROD no Gerenciador de Arquivos da loja e envie as imagens das cores. O nome do arquivo da imagem deve ser igual ao nome da cor cadastrada na loja.

Caso a cor seja composta por mais de um nome, utilize o sinal de + (mais), exemplo, "Azul+Escuro" e o nome da imagem enviada para a pasta CORES deve ser "Azul_Escuro.gif", substitua o sinal de + (mais) por _ (underscore / underline).
