Parâmetro Padrão Opções Descrição
myOptions
autoSelect true true/ false Selecione automaticamente o primeiro subproduto caso seja utilizado a opção true
cartOnPage true true/ false Ativa o recurso Adiciona produto à cesta sem sair da página na Grade caso seja utilizado a opção true
incMultGrid true true/ false Ativa o recurso para informar a quantidade de cada produto antes de colocá-lo na cesta na Grade caso seja utilizado a opção true
idElementGrid #idMainGrid id do elemento DIV Define o elemento HTML DIV a onde será inserido a Grade
separadorRGBCor #idMainGrid | Define o caractere para seperar o nome e o valor Hexadecimal da cor cadastrado no produto
qtyDescriptors 2 números entre 1 e 7 Define a quantidade máxima de descritores especiais e simples utilizandos nos produtos, incluí também a cor
htmlFlagChecked <i class="FCCheckedGrid"</i> html Define o HTML para exibir o check
imageProduct cor nome de um descritor ex.: cor e descritor1 Define qual descritor será usado para exibir a imagem dos subprodutos ex.
colorImgFormat .gif extensões de arquivos de imagem ex. .svg e .png Define a extensão dos arquivos de imagem que serão para as cores ex.
order ['cor', 'descritor1', 'descritor2', 'descritor3', 'descritorD1', 'descritorD2', 'descritorD3'] Define a ordenação que os descritores serão exibidos na Grade
myOptions : nameDescriptor [ ]
cor Selecione texto livre, ex. selecione a cor Define o texto que será exibido para a Cor
descritor1 Selecione texto livre, ex. selecione o tamanho Define o texto que será exibido para o Descritor Adicional 1
descritor2 Selecione texto livre Define o texto que será exibido para o Descritor Adicional 2
descritor3 Selecione texto livre Define o texto que será exibido para o Descritor Adicional 3
