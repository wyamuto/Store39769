Tags especiais para ConfirmacaoPedido.htm

O arquivo ConfirmacaoPedido.htm pode conter as seguintes tags especiais, que serão substituídas por conteúdo obtido a partir do pedido que acabou de ser feito pelo cliente.

As tags são acrescidas de sufixo numérico de ordem. Por exemplo, no 1º produto do pedido, as tags são <RefProduto1>, <NomeProduto1>, <QtdProduto1>, etc. No 2º produto do pedido, as tags são <RefProduto2>, <NomeProduto2>, <QtdProduto2>, etc.

A tag <PresenteProdutoX> é substituída por S caso o Xº produto tenha sido solicitado com embalagem para presente.


O arquivo ConfirmacaoPedido.htm poderá conter também as tags especiais dos arquivos htm personalizados.
