Tags especiais para Anúncios no MercadoLivre

As descrições dos anúncios publicados no MercadoLivre a partir do site administrativo da loja virtual podem conter as seguintes tags especiais. Estas tags serão substituídas por conteúdo obtido a partir do cadastro de produtos da loja:

Tag:  <NomeProd>

Substituída pelo nome do produto.

Tag:  <NomeProdP1>

Substituída pelo nome do produto até o caractere | (pipe) ou até o final, caso este caractere não exista no nome.

Tag:  <NomeProdP2>

Substituída pelo nome do produto após o caractere | (pipe) ou por branco, caso este caractere não exista no nome.

Tag:  <CodProd>

Substituída pelo código (referência) do produto.

Tag:  <Largura>, <Altura>, <Profundidade>

Substituídas respectivamente pela largura, altura e profundidade do produto, incluindo a unidade (ex: 32cm).

Tag:  <Peso>

Substituída pelo peso do produto, incluindo a unidade (ex: 2kg).

Tag:  <Cor>

Substituída pelo nome da cor do produto.

Tag:  <CorRGB>

Substituída pelo código RGB da cor do produto.

Tag:  <AdicionalD1>, <AdicionalD2>, <AdicionalD3>

Substituídas pelos descritores simples 1, 2 e 3.

Tag:  <Adicional1>, <Adicional2>, <Adicional3>

Substituídas pelos descritores especiais 1, 2 e 3.

Tag:  <LCodProd>
<LCor>
<LPeso>
<LLargura>
<LAltura>
<LProfundidade>
<LAdicionalD1>
<LAdicionalD2>
<LAdicionalD3>
<LAdicional1>
<LAdicional2>
<LAdicional3>

Substituídas pelas legendas das respectivas tags e seus valores, caso estes tenham sido preenchidos.

 

Exemplo: A tag <LLargura> será substituída pela palavra Largura mais o valor da largura (ex: Largura 35cm). A tag ficará em branco se o produto não tiver a largura preenchida.

 
 

Tag:  <IDProduto>

Substituída pelo número que identifica internamente cada produto.

Tag:  <URLPastas>

Substituída pelo endereço da pasta raiz da loja. Utilizada para criar referências para arquivos contidos nas pastas images e prod.

Exemplo de URL do logotipo da loja: <URLPastas>images/Logotopo.png

Tag:  <URLFrete>

Substituída pelo endereço para cálculo de frete do produto, já considerando seu peso. O cliente deverá informar o CEP e a quantidade desejada.

ATENÇÃO: Esta tag pode ser utilizada somente se a loja utilizar a opção Utilizar faixas de CEP (Brasil) na página Envio & frete do site administrativo.

Tag:  <URLFreteEnc>

Idêntica à tag <URLFrete>, com codificação de URL (URL Encoded). Esta tag pode ser usada para gerar o QR-Code para facilitar o acessso à página do cálculo de frete, como no exemplo abaixo:

<img src="http://chart.apis.google.com/chart?chs=165x165&cht=qr&chld=|1&chl=<URLFreteEnc>" alt="Cálculo do frete" width="165" width="165"/>

Tag:  <DescricaoCurta>

Substituída pela descrição curta do produto.

Tag:  <DescricaoLonga>

Substituída pela descrição longa do produto.

Tag:  <ImagemProd>, <ImagemDet>, <ImagemAmp>

Substituídas respectivamente pela imagem principal, de detalhe e ampliada do produto.

Tag:  <ImagemProdSub>, <ImagemDetSub>, <ImagemAmpSub>

Substituídas respectivamente pela imagem principal, de detalhe e ampliada do subproduto.

Tag:  <DescrURL>

Link externo para página com mais informações sobre o produto (exemplo: página do produto no site do fabricante).

Será utilizado o endereço preenchido no campo URL de apoio da ficha do produto no site administrativo.

ATENÇÃO: Utilize esta tag somente se o endereço apontar para uma página onde não é possível fazer a compra do produto, direta ou indiretamente. Por exemplo, a página não poderá conter o telefone da loja nem seu endereço na web. Por outro lado, o endereço poderá ser um documento .PDF online contendo o manual do produto.

Tag:  <DescrHTM>

Substituída pelo conteúdo do arquivo HTML que contém a descrição do produto. O nome deste arquivo é preenchido no campo Descrição HTML da ficha do produto.

Tag:  <MeliUserID>

Substituída pelo ID do vendedor no MercadoLivre. Utilizada para montar links para exibir mais produtos do vendedor, como no exemplo abaixo:

http://lista.mercadolivre.com.br/_CustId_<MeliUserID>

Tag:  <MeliNickname>

Substituída pelo apelido do vendedor no MercadoLivre. Utilizada para montar links para exibir perfil do vendedor e sua reputação, como no exemplo abaixo:

http://perfil.mercadolivre.com.br/<MeliNickname>

Obs1: Todas as tags são opcionais.

Obs2: As tags especiais genéricas não podem ser utilizadas.
