Tags especiais de produtos

O arquivo EstiloProduto.htm pode conter as seguintes tags especiais de produtos, que serão substituídas por conteúdo obtido a partir do cadastro de produtos da loja:

Tag especialDescrição
<ProdLista>
</ProdLista>		Delimitam o código HTML das listas de produtos.
Se a loja utiliza autopaginação, somente o conteúdo entre estas tags será repetido a cada página de produtos automaticamente carregada.
Tags obrigatórias.
<ProdDet>
</ProdDet>		Delimitam o código HTML de detalhe de cada produto.
Tags obrigatórias.
<ProdAmp>
</ProdAmp>		Delimitam o código HTML de ampliação de imagem de cada produto.
Tags obrigatórias.
<FormProd>
</FormProd>		Delimitam o posicionamento das tags de descritores, cor e de carrinho de compras.
Tags obrigatórias: deve haver um conjunto para cada tag delimitadora acima.
<Sub>
</Sub>		Delimitam o código HTML de cada subproduto. Estas tags são obrigatórias somente se a loja utiliza subprodutos. Neste caso, deve haver um conjunto para cada tag delimitadora: <ProdLista>, <ProdLista> e <ProdLista>
<ProdTop>
</ProdTop>		Delimitam o código HTML exibido no início de todas as páginas de produtos. Utilizada para incluir scripts client-side que calculam e exibem os parcelamentos disponíveis para cada produto.
<ProdBottom>
</ProdBottom>		Delimitam o código HTML exibido no final de todas as páginas de produtos. Utilizada para incluir tag </FormProdMult> para fechamento de form de múltiplos produtos (veja detalhes ao final desta página).
<WebSlice>
</WebSlice>		Delimitam o código HTML que será relacionado a um Web Slice, que é um recurso existente a partir do Internet Explorer 8 para exibir páginas pela barra de favoritos. Quando o visitante clicar no ícone verde do Web Slice, o produto será colocado na sua barra de favoritos. Ao selecionar este favorito, os dados do produto serão exibidos pelo arquivo WebSlice.htm, que deverá existir na pasta HTM da loja contendo ao menos uma tag <prod>. Quando houver alteração no preço ou na disponibilidade do produto, este favorito será exibido em negrito pelo navegador.
<JurosParcelas>Tag exclusiva para a seção <ProdTop>, que deve ser incluída caso a loja exiba os parcelamentos disponíveis para cada produto. Se esta tag existir, o arquivo JurosParcelas.js deverá ser enviado para a pasta htm da loja, contendo os juros para cada número de parcelas.
<IDLoja>Utilizada para criar links para as páginas ASP do FastCommerce.
Ex: Para criar link para a página Ajuda.asp, utilize
<a href=Ajuda.asp?IDLoja=<IDLoja>>
<IDLojaNum>Utilizada para criar formulários e links para arquivos da loja. Esta tag é substituída somente pelo ID da loja, SEM o parâmetro dinâmico Y.

Ex: Para criar formulário contendo o ID da loja, utilize

<input type=hidden name=IDLoja value=<IDLojaNum>>
<Y>Esta tag é substituída pelo valor do Y dinâmico, que varia para cada visitante. Utilizada para criar forms para as páginas ASP do FastCommerce.

Ex: Para incluir campo hidden com o Y correto, utilize

<input type=hidden name=Y value=<Y>>

A linha acima resulta em:

<input type=hidden name=Y value=1166564875496>
<BaseLoja>Utilizada para compor caminhos para as pastas da loja.
Ex: Para obter o caminho para o arquivo BotCarrinho.gif na pasta IMAGES, utilize <img src=<BaseLoja>images/BotCarrinho.gif>
<IDProduto>Substituída pelo número que identifica internamente cada produto.
<NomeProd>Nome do produto. Subprodutos utilizarão o nome do produto pai.
<NomeProdUA>Nome do produto para criar URLs amigáveis, contendo somente caracteres alfanuméricos. Os demais caracteres serão substituídos por hífens.
<CodProd>Código de referência do produto.
<BarCode>Código de barras do produto, exibido de acordo com o padrão definido na página Dados da loja.
<LegendaCat>Palavra para categorização dos produtos da loja, determinada no campo Produtos agrupados por da página Dados da loja.
Ex: categoria, departamento, coleção.
<NomeCat>Nome da categoria (ou departamento, coleção, etc) a qual pertence o produto. Veja detalhes no campo Produtos agrupados por da página Dados da loja.
<PathCat>Informa o caminho completo até a categoria (ou departamento, coleção, etc) a qual pertence o produto (ex: DVDs > Filmes > Ação). Veja detalhes no campo de subcategorias da página Dados da loja.
<ImagemProd>Substituída pela imagem do produto. A imagem exibida estará de acordo com a tag delimitadora onde esta tag está contida. Por exemplo, se a tag <ImagemProd> estiver contida entre as tags <ProdAmp> e </ProdAmp>, será exibida a imagem ampliada do produto.
<ImagemProdPri>
<ImagemProdDet>
<ImagemProdAmp>		Substituídas respectivamente pela imagem principal, de detalhe e ampliada do produto. Utilizadas na programação de scripts personalizados para exibição das imagens.
<ImagemProdPriPai>
<ImagemProdDetPai>
<ImagemProdAmpPai>		Substituídas respectivamente pela imagem principal, de detalhe e ampliada do produto-pai do subproduto. Utilizadas na programação de scripts personalizados para exibição das imagens.
<VideoProd>Substituída pelo código HTML que exibe o vídeo no YouTube associado ao produto.
<CodVideoProd>Substituída pelo código alfanumérico que identifica o vídeo no YouTube associado ao produto. Este códido permite a personalização do vídeo com mais opções oferecidas pelo YouTube ou para integrar a outro serviço de hospedagem de vídeos.
<ProdLargura>Substituída pelo valor do campo Largura máxima das fotos informado na página Dimensões & menus.
<ProdAltura>Substituída pelo valor do campo Altura mínima das fotos informado na página Dimensões & menus.
<Lancamento>Se o produto for lançamento, esta tag será substituída pela imagem BotLancamento.gif da loja.
<Destaque>Se o produto tiver o campo Destaque marcado na sua ficha, esta tag será substituída pela imagem BotDestaque.gif da loja. A imagem pode conter o texto da preferência do lojista (ex: Mais vendido), de acordo com o uso do campo Destaque (ex: para marcar os produtos mais vendidos).
<Descricao>Será substituída pela descrição longa do produto se esta for preenchida e se a tag <Descricao> estiver contida entre as tags <ProdDet> e </ProdDet>. Caso contrário será exibida a descrição curta do produto.
<Adicional1>
<Adicional2>
<Adicional3>		Substituídas pelos descritores especiais de cada produto, definidos na página Dados da loja. Uma caixa de seleção será exibida se o descritor for multi-valorado. O cliente terá que escolher uma opção para cada descritor multi-valorado para adicionar o produto ao seu carrinho.
<AdicionalD1>
<AdicionalD2>
<AdicionalD3>		Substituídas pelos descritores simples de cada produto, definidos na página Dados da loja. Uma caixa de seleção será exibida se o descritor for multi-valorado. O cliente terá que escolher uma opção para cada descritor multi-valorado para adicionar o produto ao seu carrinho.
<Cor>Substituída pelas cores de cada produto. Uma caixa de seleção será exibida se houver mais de uma cor no produto. Nesta caso, o cliente terá que escolher uma opção para adicionar o produto ao seu carrinho.
<Peso>Substituída pelo peso de cada produto.
<WidthProd>Substituída pela largura de cada produto. A unidade de largura também será exibida (ex: 25,1 cm). Acesse a página Dados da loja para definir a unidade utilizada pela loja.
<HeightProd>Substituída pela altura de cada produto (ex: 34,3 cm).
<DepthProd>Substituída pela profundidade de cada produto (ex: 20,5 cm).
<Estoque>Substituída pela quantidade em estoque de cada produto.
<LinkDet>Se o produto tiver descrição longa ou imagem de detalhe, esta tag será substituída pela imagem BotMaisDetalhes.gif da loja. Esta imagem poderá ser clicada para que o cliente veja mais detalhes sobre o produto.
<LinkAmp>Se o produto tiver imagem ampliada, esta tag será substituída pela imagem BotAmpliar.gif da loja. Esta imagem conterá link para que o cliente veja a imagem ampliada do produto.
<LinkComprar>Se o produto estiver disponível para compra, esta tag será substituída pela imagem BotCarrinho.gif da loja. Esta imagem conterá link para que o cliente inclua o produto no carrinho.
<Preco>Substituída pelo preço normal e/ou preço promocional do produto, no formato De/Por.
<PrecoSimples>Substituída pelo preço normal ou preço promocional do produto, com a legenda da moeda (R$) mas nenhuma outra formatação.
<PrecoNum>Substituída pelo preço normal ou preço promocional do produto, sem legenda da moeda (R$) ou qualquer outra formatação. Utilizada em scripts client-side que calculam e exibem os parcelamentos disponíveis para cada produto.
<PrecoOri>Substituída pelo preço normal do produto (fora do período promocional), ou pelo preço promocional do produto caso o preço normal não esteja preenchido na ficha do produto, sem legenda da moeda (R$) ou qualquer outra formatação. Utilizada em scripts client-side que calculam e exibem o valor ou o percentual economizado pelo cliente na compra do produto.
<MaxParcelasProd>Informa o número máximo de parcelas, conforme preenchido no campo Máx Parc de cada produto. Se este campo contiver 1, esta tag será substituída por à vista. Se contiver 0, a tag será removida.
<MaxParcelasProdNum>Informa o número máximo de parcelas do produto, conforme preenchido no campo Máx Parc de cada produto. Utilizada em scripts client-side que calculam e exibem os parcelamentos disponíveis para cada produto.
<DataPromFim>Substituída pela data/hora de término da validade do preço promocional de cada produto.
<CrossSelling>Substituída pelos produtos relacionados a este produto, no campo de cross selling da sua ficha cadastral. Esta tag é utilizada somente se o campo Usar tag <CrossSelling> da página Textos & imagens estiver marcado. Caso contrário, a tag será ignorada e os produtos relacionados serão exibidos ao final da página com os detalhes do produto.
<Comments>Substituída pelos comentários feitos pelos clientes sobre este produto. Esta tag é utilizada somente se o campo Usar tag <Comments> da página Dados da loja estiver marcado. Caso contrário, a tag será ignorada e os comentários serão exibidos ao final da página com os detalhes do produto.
<OpiniaoMedia>Substituída pela média das avaliações sobre os produtos feitas pelos clientes, liberadas até a véspera. Normalmente a nota é indicada com estrelas.
<OpiniaoMediaNum>Substituída pelo valor médio das avaliações sobre os produtos feitas pelos clientes, liberadas até a véspera. O valor é arredondado na 3ª casa decimal.
<OpiniaoQtd>Substituída pela quantidade de avaliações sobre os produtos feitas pelos clientes, liberadas até a véspera.
<LinkOpiniao>Substituída por link para que o cliente opine sobre o produto. Será necessário selecionar opção ativa para o campo Avaliação pelo cliente na página Dados da loja.
<AvisoDisp>Caso o produto esteja esgotado, esta tag será substituída por link para que o cliente solicite para ser avisado automaticamente quando o produto estiver novamente disponível. Será necessário selecionar o campo Aviso de disponibilidade na página Dados da loja.
<IndicaProd>Substituída por link para que o cliente recomende este produto a terceiros por e-mail. Será necessário selecionar o campo Link de recomendação na página Estilos & cores.
<DescrURL>Substituída por link para página que contém mais informações sobre o produto, no site do fabricante ou em página especial na própria loja, conforme informado no cadastro do produto. Será necessário selecionar o campo Texto para link de mais informações na página Textos & imagens.
<DescrHTM>Substituída pelo conteúdo do arquivo HTM informado no campo Descrição HTML da ficha do produto.
<IncludeHTML> Substituída pelo arquivo .htm ou .html indicado no atributo src= da tag. Este arquivo deve existir na pasta HTM ou subpasta desta.

Exemplo:

<IncludeHTML src="outros/more.htm">

Opcionalmente, pode-se utilizar os atributos param1= até param9=. Estes atributos serão substituídos respectivamente pelas tags <param1> até <param9>, caso estas existam nos arquivo incluídos.

Exemplo:

<IncludeHTML src="teste.htm" param1="Vitrine especial" param2="50" param3="#BB0000">

Os arquivos incluídos podem conter tags <IncludeHTML>, ou seja, é possível fazer inclusões dentro de inclusões, de forma recursiva. No total, até 5 tags <IncludeHTML> serão substituídas por arquivo e suas inclusões.
<EstiloSubProduto>Substituída alternadamente pelas classes CSS EstCorParSubProd e EstCorImparSubProd. Esta tag permite alternar as cores de subprodutos adjacentes em um mesmo produto.
<Vakinha>Substituída por link para que o visitante possa criar uma vaquinha para comprar o produto. Veja mais informações sobre o Vakinha na página Pagamentos.


Tags para contextualização e refinamento de buscas:
As seguintes tags são exclusivas para a seção <ProdTop> e são utilizadas para exibição de conteúdo referente aos filtros de busca da página atual.

Tag especialDescrição
<QtProd>Indica a quantidade de produtos encontrados. Não são contabilizados os subprodutos.
<LinkAdvancedSearch>Substituída por link para a página de busca detalhada de produtos, utilizando os filtros da página atual.
<ListPathSearch>Substituída pelo caminho completo da página atual de produtos, com todos os filtros de busca utilizados, incluindo os termos digitados pelo visitante (ex: DVDs > Filmes > Ação > "tom cruise"). Ao lado de cada filtro, será exibido um ícone para removê-lo. Ao clicar no termo de busca, será possível digitar outro termo para fazer nova busca utilizando os demais filtros. Ao clicar nos demais filtros, será feita busca exclusiva pelo filtro clicado.
<ListPathCatQt>Lista as quantidades de produtos encontrados em cada categoria dos produtos listados na página atual.
<ListAdic1Qt>
<ListAdic2Qt>
<ListAdic3Qt>		Lista as quantidades de produtos encontrados em cada descritor especial dos produtos listados na página atual.
<NomeAdicional1>
<NomeAdicional2>
<NomeAdicional3>		Substituídos pelos nomes dos descritores especiais 1, 2 e 3.

Exemplos: Tamanho, Marca, Voltagem
<LegendaCat>
<LegendaCats>		Substituídas pela legenda escolhida para representar as categorias da loja, no campo Produtos agrupados por da página Dados da loja.
Exemplos: Departamento, Seção

A tag <LegendaCats> é substituída pela legenda no plural.
Exemplos: Departamentos, Seções


Tags para grades de produtos:
Utilizadas por scripts para a escolha de tamanhos e cores de subprodutos.

Tag especialDescrição
<NomeAdicional1>
<NomeAdicional2>
<NomeAdicional3>		Substituídas pelos nomes dos descritores especiais 1, 2 e 3, cadastrados nos três campos de descritores especiais da página Dados da loja.
<NomeAdicionalD1>
<NomeAdicionalD2>
<NomeAdicionalD3>		Substituídas pelo nomes dos descritores simples 1, 2 e 3, cadastrados nos três campos de descritores simples da página Dados da loja.
<Adicional1Val>
<Adicional2Val>
<Adicional3Val>		Substituídas pelos valores dos descritores especiais 1, 2 e 3 cadastrados na ficha do subproduto.
<AdicionalD1Val>
<AdicionalD2Val>
<AdicionalD3Val>		Substituídas pelos valores dos descritores simples 1, 2 e 3 cadastrados na ficha do subproduto.
<CorVal>Substituída pelo nome da cor cadastrada na ficha do subproduto.
<CorRGB>Substituída pelo valor RGB da cor cadastrada na ficha do subproduto.
<PesoNum>Substituída pelo valor do peso de cada produto. Utilizada também em scripts client-side que utilizam o peso do produto (por exemplo, para exibir promoções de frete grátis se o peso cadastrado for zero).
<WidthProdNum>Substituída pelo valor da largura de cada produto.
<HeightProdNum>Substituída pelo valor da altura de cada produto.
<DepthProdNum>Substituída pelo valor da profundidade de cada produto.

Obs1: Todas as tags são opcionais, exceto os quatro primeiros conjuntos de tags delimitadoras.

Obs2: As tags especiais genéricas não podem ser utilizadas no arquivo EstiloProduto.htm.


Tags para adicionar múltiplos produtos e quantidades à cesta:

As seguites tags são necessárias para que as páginas de produtos exibam campos de quantidade e seleção e possam enviar os produtos selecionados simultaneamente para a cesta.

Tag especialDescrição
<FormProdMult>Deve ser inserida dentro da tag <ProdTop>.
</FormProdMult>Deve ser inserida dentro da tag <ProdBottom>.
<IncMult>Substituída por caixa de seleção (checkbox) e campo de texto para informar a quantidade desejada do produto.

Para colocar apenas um link de comprar ao final da página, inclua a tag <LinkComprar> somente na seção <ProdBottom>. Adicionalmente, deve-se remover todas as ocorrências das tags <FormProd> e </FormProd>.


Tags especiais para redes sociais:

Para utilizar estas tags, é necessário marcar o campo Usar tags especiais da seção de redes sociais da página Dados da loja.

Tag Descrição
<ShareFacebook> Substituída pelo ícone pequeno do Facebook, que poderá ser clicado pelo visitante para compartilhar o produto no seu perfil no Facebook.
<ShareTwitter> Substituída pelo ícone pequeno do Twitter, que poderá ser clicado pelo visitante para twittar o produto em seu profile no Twitter.
<GooglePlusOne> Substituída pelo código do Google que exibirá botão para que o visitante possa recomendar o produto nos resultados de busca do Google. É possível incluir parâmetros para alterar o tamanho do botão e exibir a quantidade de recomendações do produto.

Exemplos:

<GooglePlusOne size="small">: exibe o botão no menor formato, com contador de recomendações.

<GooglePlusOne size="medium" count="false">: exibe o botão no tamanho médio, sem contador de recomendações.
<Facebook type="plugin"> Substituída pelo código do Facebook que exibirá um social plugin do Facebook. Estes plugins são utilizados para que o visitante possa comentar, recomendar, curtir ou enviar o produto para seu perfil no Facebook.

Plugins mais utilizados em produtos:

<Facebook type="comments">: exibe os comentários feitos pelos visitantes sobre o produto. Estes comentários podem ser moderados pela loja, se o campo Facebook AppID da página Dados da loja estiver preenchido e a aplicação estiver corretamente configurada no Facebook.

<Facebook type="like">: exibe o botão Curtir. Esta tag pode também ser configurada para exibir o botão Recomendar (veja abaixo nos exemplos de parâmetros).

<Facebook type="send">: exibe o botão Enviar. Esta tag permite ao visitante enviar um resumo do produto para um amigo no Facebook.

É possível incluir parâmetros para configurar os recursos de cada plugin.

Exemplos:

<Facebook type="comments" width="700" num_posts="5">
Exibe os 5 posts mais recentes em uma caixa de 700 pixels de largura.

<Facebook type="like" send="true" layout="button_count" show_faces="false">
Exibe botão Curtir com contador e sem exemplificar nomes de pessoas que curtiram o produto.
