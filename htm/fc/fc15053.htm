Para exportar ofertas para o Walmart


No menu esquerdo do site administrativo da sua loja virtual no Fastcommerce, clique no item MARKETPLACES > Walmart e depois no botão Exportar.


A página de exportação é dividida em três seções:

  • Filtro de produtos: Nesta seção serão informados os critérios de seleção dos produtos que serão exportados.
  • Dados das novas ofertas: as ofertas exportadas serão criadas com as informações contidas nesta seção, incluindo nome, imagens etc.
  • Descrições das ofertas: o design padronizado das ofertas exportadas é definido nesta seção.


O campo Multiplicador do preço pode ser utilizado para alterar os preços dos produtos exportados. Se este campo for preenchido com valor diferente de 1, as ofertas criadas terão preços diferentes dos produtos da loja. Por exemplo, para oferecer um desconto de 30% nas ofertas, preencha este campo com 0,7. Para aumento de 15%, preencha este campo com 1,15.

A quantidade e as dimensões são campos opcionais. Se a quantidade não for informada, será utilizada a quantidade em estoque de cada produto.

Marque os itens da seção Especificações que deseja que sejam informados nas ofertas exportadas.

Nos campos de sincronismo, informe as direções de sincronismo desejadas quando ocorrem alterações nos estoques e nos preços dos produtos.

Após preencher os campos desejados, clique no botão Iniciar exportação para enviar as ofertas para o Walmart, que posteriormente serão avaliadas e publicadas pela equipe do marketplace.
