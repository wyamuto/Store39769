(*) A imagem principal é apresentada na página inicial da loja e na listagem dos produtos.

(*) A imagem de detalhe é apresentada ao abrir as informações de um produto.

(*) A imagem ampliada é apresentada quando o usuário passa o mouse sobre a imagem de detalhe (recurso de zoom).

Os campos referentes às imagens devem ser preenchidos conforme os exemplos abaixo:

Para produto que não possui subprodutos:


Imagem principal:

produtoangulo1.jpg

Imagem de detalhe:

produtoangulo1.jpg,produtoagulo2.jpg

Imagem ampliada:

produtoangulo1.jpg,produtoangulo2.jpg

Para produto que possui subprodutos:


Produto principal

Imagem principal:

produtoangulo1.jpg

Sub-produto

Imagem de detalhe:

produtoangulo1.jpg,produtoagulo2.jpg

Imagem ampliada:

produtoangulo1.jpg,produtoangulo2.jpg

Cada produto deve possuir uma imagem principal. Caso seja necessário cadastrar várias imagens, preencha os campos referente às imagens de detalhe e ampliada com os nomes iguais aos dos arquivos enviados na pasta prod separados por vírgula (sem espaços antes ou depois das mesmas).

Qual é o tamanho ideal para as imagens?


Caso a loja utilize nossos modelos de design, os tamanhos recomendados para as imagens são:

Imagem principal: 200x200px
Imagem de detalhe: 400x400px
Imagem ampliada: 800x800px
