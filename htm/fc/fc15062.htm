Para habilitar este recurso, acesse Dados da loja > Produtos e marque o campo Lista de desejos.

Para que o botão Adicionar na lista de desejos seja exibido no produto, os arquivos de design da loja devem estar preparados para exibir este botão.



Existem duas opções de implementar este recurso:



1ª Opção

Utilizando a tag especial <WishList>

Esta opção é recomendada para quem deseja que o cliente insira apenas os produtos principais na lista de desejos e não os subprodutos (variações da grade) e para lojas que não tenham produtos com variações.

Primeiramente, insira a seguinte linha no arquivo LojaLib.js:

var bBuyWishlist=false;

A linha acima desativa o botão comprar direto na lista de desejos. Para comprar, o cliente deverá clicar em mais detalhes para ver o produto e escolher a variação desejada.

1.1 Para inserir na lista e no detalhe de produtos, altere o arquivo EstiloProduto.htm

Para inserir na lista, a tag <WishList> deve estar dentro de <ProdLista>...</ProdLista>.

Para inserir no detalhe, a tag <WishList> deve ser inserida dentro de <ProdDet>...</ProdDet>.

1.2 Para inserir no layout de produtos da home o arquivo é o EstiloProdutoOut.htm

A tag <WishList> deve ser inserida dentro do layout desejado. Exemplo: <p255>...<WishList>......</p255>



2ª Opção

Lista da desejos na grade de produtos

Esta opção é recomendada para quem deseja que o cliente escolha variações dos produtos na lista de desejos. Neste caso, o cliente pode comprar o produto posteriormente direto da lista de desejos.

A implementação via alteração de grade não tem um passo a passo, pois depende do tipo de grade que a loja utiliza. Para este tipo de implementação o arquivo javascript que exibe a grade deve ser alterado. Para mais informações, entre em contato com nossa equipe de suporte.
