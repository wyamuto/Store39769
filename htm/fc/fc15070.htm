Os produtos que serão exibidos no cross selling serão selecionados de acordo com o campo Cross selling preenchido na ficha de cada produto adicionado ao carrinho.


Observações


  • Se a loja não tiver subprodutos, o botão Adicionar incluirá o produto diretamente no carrinho. Caso a loja tenha subprodutos, será aberto o produto no formato de espiadinha, para que o cliente possa comprar selecionando cor, tamanho etc. Neste caso a loja deve ter o arquivo estiloproduto9.htm com o design específico dessa página na pasta HTM do gerenciador de arquivos da loja.

  • Se a resolução horizontal do navegador for menor do que 400 pixels ou se estiver na versão para celular, ao invés de abrir a espiadinha o botão "adicionar" levará à página de detalhe deste produto.

  • Após ativar o recurso acesse a loja para ver como fica a exibição desses produtos de cross selling no carrinho de sua loja. Clique no link de cada produto, no botão comprar etc. Qualquer dúvida entre em contato com a nossa equipe de suporte.





Personalizações

Ao ativar o recurso, os produtos serão exibidos com um design padronizado. Veja abaixo como personalizar esta exibição:

Configurações do design

Se desejar que no design não seja exibido alguns elementos, insira a respectiva linha indicada abaixo no arquivo lojalib.js da pasta HTM da loja:


Elemento
Configuração
Descritores
var bShowDescritCrossCart=false;
Filtros
var bShowFilCrossCart=false;
Nome da categoria
var bShowCatCrossCart=false;
Referência/código do produto
var bShowRefCrossCart=false;
Botão de carrinho de comprar
var bBuyWishlistCrossCart=false;


Personalização do CSS

No design padrão é inserido um arquivo CSS que define cores, tamanhos de textos, posicionamento etc, definindo o design do cross selling. Se desejar enviar o seu próprio CSS e que este CSS padrão não seja carregado, inclua a linha abaixo no arquivo lojalib.js:

var bIncludeCSSCrossSellingCart=false; //para remover include de css padrão

Desta forma é possível inserir no CSS da loja (lojasite.css) os estilos personalizados para obter o design desejado.


Personalização completa

Na página do carrinho da loja é inserido um objeto JSON chamado FCLib$.CrossSellingCart com os dados dos produtos de cross selling da cesta. Uma função javascript se encagerrega de ler esse objeto JSON e dinamicamente montar o design dos produtos para serem exibidos no cross selling, carregar um CSS com os estilos e inserir na página os produtos.

É possível personalizar completamente esse recurso usando uma função personalizada chamada FuncCrossSellingCart(). Basta criar uma função com esse nome no arquivo lojalib.js da loja e através dessa função manipular o objeto JSON FCLib$.CrossSellingCart que contém as informações necessárias para montar o design com os produtos de cross selling.
