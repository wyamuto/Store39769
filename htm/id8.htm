Antes de inserir o novo design sugerimos que efetue backup das pastas htm e images.

Para efetuar o backup entre no item Arquivos (canto superior direito), em seguida, selecione a pasta desejada.

Utilize a opção para marcar todos ao final da página e selecione o item Baixar marcados ao final da página para receber o arquivo ZIP com os arquivos.

Para enviar os arquivos html acesse o item Arquivos (canto superior direito) e dentro da pasta htm efetue o envio do arquivo htm8.zip marcando a opção Sobrescrever se arquivo já existir no site.

Para enviar os arquivos de imagens acesse o item Arquivos (canto superior direito) e dentro da pasta images efetue o envio do arquivo images8.zip marcando a opção Sobrescrever se arquivo já existir no site.
